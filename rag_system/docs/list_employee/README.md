# Документация сотрудников железнодорожного транспорта

## Обзор

Данный раздел содержит полную документацию о сотрудниках железнодорожного транспорта, их должностях, обязанностях, квалификационных требованиях и взаимодействии.

#### Должности
- Руководящие: начальники, заместители
- Оперативные: диспетчеры, операторы
- Технические: машинисты, монтёры, инженеры
- Медицинские: врачи, фельдшеры, медсёстры
- Охранно-полицейские: оперуполномоченные, патрульные

#### Техника
- Краны: КС-5371, КС-55713, РДК-25
- Экскаваторы: Э-153, JCB 3CX
- Бульдозеры: Т-170
- Погрузчики: Амкодор-332, ТО-28
- Автомобили: КАМАЗ-5320
- Дрезины: АГВ
- Мотовозы: МПТ
- Путевая техника: УК-25, ЩОМ-4, ВПР-02, Б-М
- Техника для контактной сети: АГ-60, МП
- Мостостроительная техника: МУ, КМ
- Средства связи: Motorola GP-340, Thuraya
- Беспилотные системы: DJI Matrice 30T, Autel EVO Max 4T
- Медицинский транспорт: реанимобили, санитарные автомобили

## Обновление документации

Документация должна обновляться:
- При изменении должностного состава
- При внедрении новых должностей
- При изменении квалификационных требований
- После крупных аварий для анализа опыта
- Не реже 1 раза в год для актуализации данных

## Контакты

За актуализацией документации отвечает:
- ДАВС — координация обновлений
- Каждое подразделение — актуализация своих разделов
