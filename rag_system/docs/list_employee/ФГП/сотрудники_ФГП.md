# Сотрудники ФГП — Федеральное государственное предприятие 'Ведомственная охрана железнодорожного транспорта Российской Федерации'

## Обзор

Данный документ содержит информацию о должностях и сотрудниках Федерального государственного предприятия 'Ведомственная охрана железнодорожного транспорта Российской Федерации' — подразделения, отвечающего за охрану железнодорожных объектов и грузов.

---

## Должности ФГП

### 8.1 Начальник караула
- **Подразделение**: ФГП
- **Квалификация**: Высшее образование, опыт работы не менее 3 лет
- **Обязанности**:
  - Руководство караулом
  - Обеспечение охраны объекта
  - Контроль действий нарядов
- **Используемая техника**:
  - Рация Motorola GP-340
  - Ноутбук
  - СИЗ (каска, жилет)
- **Примеры использования**:
  - Руководство охраной станции
  - Координация действий нарядов
  - Контроль доступа на объект

### 8.2 Ведомственный охранник
- **Подразделение**: ФГП
- **Квалификация**: Среднее образование, специальная подготовка
- **Обязанности**:
  - Охрана железнодорожных объектов
  - Контроль доступа
  - Проверка документов
- **Используемая техника**:
  - Рация Motorola GP-340
  - Металлодетектор
  - СИЗ (каска, жилет)
- **Примеры использования**:
  - Охрана станции
  - Контроль доступа на перроны
  - Проверка документов пассажиров

### 8.3 Сотрудник кинологической службы
- **Подразделение**: ФГП
- **Квалификация**: Среднее образование, специальная подготовка
- **Обязанности**:
  - Охрана с собакой
  - Обыск грузов
  - Патрулирование территории
- **Используемая техника**:
  - Рация Motorola GP-340
  - Служебная собака
  - СИЗ (каска, жилет)
- **Примеры использования**:
  - Обыск грузов на наличие запрещённых предметов
  - Патрулирование территории станции
  - Охрана вагонов с ценными грузами

## Организационная структура

```
Начальник отряда ФГП
├── Заместители
├── Караульная служба
│   ├── Начальники караулов
│   └── Ведомственные охранники
└── Кинологическая служба
    └── Сотрудники с собаками
```

---

## Требования к персоналу

### Общие требования
- Готовность к работе в любое время суток
- Физическая выносливость
- Знание правил техники безопасности
- Отсутствие судимости

### Специальные требования
- Наличие лицензии на охранную деятельность
- Прохождение медицинского осмотра
- Обучение по программе охраны труда

---

## Режим работы

- **Дежурный персонал**: Круглосуточное дежурство (график 2/2)
- **Оперативный персонал**: Круглосуточное дежурство
- **Кинологическая служба**: График 5/2 с возможностью вызова

## Взаимодействие с другими подразделениями

ФГП взаимодействует со следующими подразделениями:
- **ДАВС** — охрана аварийно-восстановительных работ
- **ЦД** — охрана диспетчерских пунктов
- **ЦЧС** — охрана ситуационного центра
- **МЧС** — координация охраны при ЧС
