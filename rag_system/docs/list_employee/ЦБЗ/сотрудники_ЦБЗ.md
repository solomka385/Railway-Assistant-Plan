# Сотрудники ЦБЗ (Департамент безопасности ОАО «РЖД»)

## Общая информация

Департамент безопасности ОАО «РЖД» (ЦБЗ) — структурное подразделение, обеспечивающее комплексную безопасность деятельности ОАО «РЖД», защиту имущества и персонала.

## Организационная структура

### Руководство
- **Директор Департамента безопасности** — возглавляет департамент, определяет политику безопасности
- **Заместители директора** — курируют отдельные направления деятельности
- **Начальники управлений** — руководят профильными управлениями

### Управления департамента
- **Управление экономической безопасности** — предотвращает экономические преступления
- **Управление физической защиты** — обеспечивает охрану объектов
- **Управление информационной безопасности** — защищает информационные ресурсы
- **Управление кадровой безопасности** — проводит проверку персонала

### Оперативный персонал
- **Начальник оперативного отдела** — руководит оперативной работой
- **Старший оперуполномоченный** — проводит оперативные мероприятия
- **Оперуполномоченный** — выполняет оперативные задачи
- **Инспектор безопасности** — проводит проверки и расследования

### Технический персонал
- **Инженер по техническим средствам защиты** — обслуживает системы безопасности
- **Администратор систем безопасности** — управляет программными комплексами

## Требования к сотрудникам

### Квалификационные требования
- Высшее образование (юридическое, техническое, экономическое)
- Опыт работы в сфере безопасности не менее 3 лет
- Отсутствие судимости
- Наличие допуска к гостайне

### Профессиональные навыки
- Знание законодательства в области безопасности
- Навыки проведения оперативно-розыскных мероприятий
- Умение выявлять и предотвращать угрозы
- Навыки работы с системами безопасности

## График работы

- **Режим работы**: пятидневная рабочая неделя
- **Часы работы**: 9:00 - 18:00
- **Дежурства**: круглосуточное дежурство оперативного персонала

## Взаимодействие с другими подразделениями

### Основные контакты
- **ФГП** — ведомственная охрана
- **РЦБЗ** — региональный центр безопасности
- **ЦРБ** — Департамент безопасности движения
- **ДАВС** — координация при авариях
- **Все дирекции** — обеспечение безопасности

## Обязанности при ЧС

1. Координация действий служб безопасности при ЧС
2. Оценка угроз и рисков безопасности
3. Взаимодействие с правоохранительными органами
4. Обеспечение охраны объектов и грузов
5. Проведение расследований инцидентов
6. Разработка мер по предотвращению ЧС

## Полномочия

- Проверка деятельности подразделений по вопросам безопасности
- Проведение оперативно-розыскных мероприятий
- Временное ограничение доступа к объектам
- Привлечение специалистов к расследованиям
