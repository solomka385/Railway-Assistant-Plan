# Сотрудники МАБ (Медицинская выездная врачебно-аварийная бригада)

## Общая информация

Медицинская выездная врачебно-аварийная бригада (МАБ) — специализированное подразделение для оказания экстренной медицинской помощи при авариях и чрезвычайных ситуациях на железнодорожном транспорте.

## Организационная структура

### Руководство
- **Начальник МАБ** — отвечает за организацию работы бригады, координацию действий при ЧС
- **Заместитель начальника МАБ** — замещает начальника в его отсутствие

### Медицинский персонал
- **Врач-реаниматолог** — проводит реанимационные мероприятия, принимает решения о тактике лечения
- **Врач-травматолог** — специализируется на лечении травм и переломов
- **Врач-хирург** — выполняет экстренные хирургические вмешательства
- **Фельдшер** — оказывает первую медицинскую помощь, проводит диагностику
- **Медицинская сестра** — ассистирует врачу, выполняет процедуры, ведет документацию

### Водительский состав
- **Водитель санитарного автомобиля** — обеспечивает доставку бригады к месту происшествия

## Требования к сотрудникам

### Квалификационные требования
- Высшее медицинское образование для врачей
- Среднее медицинское образование для фельдшеров и медсестер
- Наличие действующей лицензии на медицинскую деятельность
- Опыт работы в экстренной медицине не менее 3 лет

### Профессиональные навыки
- Владение методами реанимации и интенсивной терапии
- Навыки оказания первой медицинской помощи при травмах различной тяжести
- Умение работать в стрессовых ситуациях
- Знание протоколов оказания экстренной медицинской помощи

## График работы

- **Режим дежурств**: круглосуточное дежурство по графику
- **Продолжительность смены**: 12 часов
- **График сменности**: 2/2 (два дня работы, два дня отдыха)

## Взаимодействие с другими подразделениями

### Основные контакты
- **МЧС** — совместная работа при ликвидации последствий ЧС
- **НУЗ/ЧУЗ** — госпитализация пострадавших
- **ДАВС** — координация действий при авариях на путях
- **ДЦУП** — получение информации о происшествиях

## Обязанности при ЧС

1. Немедленное выезд на место происшествия после получения сигнала
2. Оказание экстренной медицинской помощи пострадавшим
3. Тriage (сортировка) пострадавших по степени тяжести
4. Принятие решения о необходимости госпитализации
5. Сопровождение пострадавших в медицинское учреждение
6. Ведение медицинской документации

## Специальное оборудование

- Реанимобили с полным комплектом медицинского оборудования
- Дефибрилляторы
- Аппараты искусственной вентиляции легких
- Мониторы пациентов
- Наборы для экстренных операций
- Медикаменты для оказания экстренной помощи
