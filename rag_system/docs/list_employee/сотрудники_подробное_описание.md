# Подробный справочник должностей и сотрудников железнодорожного транспорта

## Обзор документа

Данный документ содержит подробное описание должностей и сотрудников железнодорожного транспорта, с указанием подразделений, обязанностей, необходимой квалификации и используемой техники.

## 1. Должности ДАВС (Дирекция аварийно-восстановительных средств)

### 1.1 Начальник АВП
- **Подразделение**: ДАВС
- **Квалификация**: Высшее техническое образование, опыт работы не менее 5 лет
- **Обязанности**:
  - Руководство аварийно-восстановительным поездом
  - Координация действий бригады при авариях
  - Планирование восстановительных работ
  - Обеспечение безопасности персонала
- **Используемая техника**:
  - Рация Motorola GP-340
  - Спутниковый телефон Thuraya
  - Ноутбук с картами местности
- **Примеры использования**:
  - Руководство работами при сходе подвижного состава
  - Координация действий при обрушении мостов
  - Планирование восстановительных работ

Данный справочник содержит подробное описание должностей и сотрудников железнодорожного транспорта, с указанием подразделений, обязанностей, необходимой квалификации и используемой техники.

Информация может использоваться для:
- Планирования штатного расписания
- Обучения персонала
- Разработки регламентов взаимодействия
- Оптимизации использования кадров
