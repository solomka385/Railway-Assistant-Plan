# Техника ДГПС — старший дорожный диспетчер

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДГПС (старший дорожный диспетчер) для оперативного управления движением и принятия решений при авариях.

## Назначение

Оперативное управление движением, принятие решений при авариях.

## Технические средства

### АРМ диспетчера
- Количество мониторов: 4
- Программное обеспечение:
  - "Экспресс-М" — система управления движением поездов
  - "Мониторинг-2025" — система мониторинга состояния инфраструктуры
- Резервное питание: ИБП на 8 часов

### Система GPS-трекинга локомотивов
- Обновление данных: каждые 10 сек
- Отображение в реальном времени на АРМ диспетчера
- Интеграция с системами управления движением

### Радиостанции
- Модель: Sepura SRH3800
- Диапазон: УКВ (160 МГц)
- Стандарт: цифровой DMR
- Назначение: связь с машинистами и бригадами

### Система оповещения пассажиров
- Каналы оповещения:
  - Громкоговорители на станциях
  - SMS-оповещения
  - Информационные табло
- Автоматическое оповещение при задержках и изменениях

### Резервное оборудование
- Источник бесперебойного питания (ИБП) на 8 часов
- Резервные каналы связи
- Дублирующие рабочие места

## Технические характеристики

**Ежедневно:**
- Проверка работоспособности всех мониторов
- Проверка запуска программного обеспечения
- Проверка связи с системами мониторинга
- Проверка резервного питания

**Еженедельно:**
- Тестирование систем оповещения
- Проверка резервных каналов связи
- Обновление карт и схем

**Ежемесячно:**
- Полная диагностика оборудования
- Обновление программного обеспечения
- Проверка систем резервирования

### Регламент эксплуатации радиостанций

**Ежедневно:**
- Проверка заряда аккумуляторов
- Проверка чистки контактов
- Тестирование связи

**Еженедельно:**
- Полная проверка работоспособности
- Проверка антенн

## Примеры применения

**11 ноября 2024, станция "Люберцы-1"**
Пожар в здании вокзала. ДГПС перенаправил 12 поездов, задействовал МЧС, ФГП. Восстановление за 3 часа.

При возникновении пожара диспетчер оперативно:
- Остановил движение на подходах к станции
- Перенаправил поезда по альтернативным маршрутам
- Вызвал аварийные службы (МЧС, ФГП)
- Организовал оповещение пассажиров

## Взаимодействие с другими подразделениями

Техника подразделения ДГПС взаимодействует с техникой других подразделений при:
- Оперативном управлении с ЦУП (Центр управления перевозками)
- Координации с ЦД (Центральная дирекция управления движением)
- Взаимодействии с ДЦС (Центр организации работы станций)
- Работе с ДЦУП (Диспетчерский центр управления перевозками)
- Аварийных ситуациях с МЧС
- Охране объектов с ФГП
