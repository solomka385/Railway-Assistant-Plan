# Техника РЦБЗ — Московский региональный центр безопасности

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением РЦБЗ (Московский региональный центр безопасности) для физической безопасности объектов и контроля доступа.

## Назначение

Физическая безопасность объектов, контроль доступа.

## Технические средства

### IP-камеры Hikvision с ИИ-аналитикой
- Характеристики:
  - Разрешение: до 4K
  - ИИ-распознание лиц
  - Распознавание подозрительных действий
  - Ночное видение
- Назначение: видеонаблюдение за объектами

### Дроны DJI с тепловизорами
- Характеристики:
  - Время полёта: до 40 мин
  - Радиус действия: до 10 км
  - Тепловизор для ночного патрулирования
- Назначение: патрулирование периметра

### Мобильные КПП
- Оснащение:
  - Рамки металлодетекторов
  - Биометрический контроль
  - Системы видеонаблюдения
  - Средства связи
- Назначение: контроль доступа на объекты

### Системы контроля доступа
- Характеристики:
  - RFID-карты
  - Биометрическая идентификация
  - Автоматические турникеты
- Назначение: контроль доступа на объекты

### Патрульные автомобили УАЗ Patriot
- Характеристики:
  - Полный привод
  - GPS-трекинг
  - Системы связи
- Назначение: патрулирование территории

### Специальное оборудование
- Системы видеонаблюдения
- Системы сигнализации
- Средства связи
- Оборудование для работы с документами

## Технические характеристики

### Регламент эксплуатации IP-камер

**Ежедневно:**
- Проверка записи видео
- Проверка чистки объективов
- Проверка работоспособности ИИ-аналитики

**Еженедельно:**
- Полная проверка камер
- Проверка настроек распознавания
- Тестирование системы оповещения

**Ежемесячно:**
- Обновление прошивки камер
- Проверка систем хранения данных
- Техническое обслуживание

### Регламент эксплуатации дронов

**Перед вылетом:**
- Проверка состояния дрона
- Проверка тепловизора
- Проверка заряда аккумуляторов

**Во время полёта:**
- Контроль за качеством видео
- Проверка работоспособности оборудования

**После полёта:**
- Очистка оборудования
- Проверка на наличие повреждений
- Зарядка аккумуляторов

**Техническое обслуживание:**
- Плановое ТО — ежемесячно
- Калибровка тепловизора — ежеквартально

### Регламент эксплуатации мобильных КПП

**Перед использованием:**
- Проверка работоспособности рамок металлодетекторов
- Проверка биометрического контроля
- Проверка систем видеонаблюдения

**Во время работы:**
- Контроль за работой систем
- Проверка систем безопасности

**После использования:**
- Проверка состояния оборудования
- Запись в журнал

## Регламент эксплуатации и обслуживания

### Системы видеонаблюдения
- Ежедневная проверка работоспособности
- Еженедельное обслуживание камер
- Ежемесячное обновление прошивки
- Техническое обслуживание — ежемесячно

### Дроны
- Ежедневная проверка работоспособности
- Плановое ТО — ежемесячно
- Калибровка тепловизора — ежеквартально

### Системы контроля доступа
- Ежедневная проверка работоспособности
- Еженедельное тестирование
- Техническое обслуживание — ежемесячно

### Патрульные автомобили
- Ежедневная проверка технического состояния
- Плановое ТО — каждые 5000 км
- Текущий ремонт — по необходимости

## Примеры применения

**Попытка поджога на станции**
Камеры зафиксировали — охрана задержала злоумышленника.

При попытке поджога были задействованы:
- IP-камеры с ИИ-аналитикой для фиксации нарушения
- Дроны с тепловизорами для патрулирования
- Патрульные автомобили для задержания
- Системы контроля доступа для предотвращения доступа

## Взаимодействие с другими подразделениями

Техника подразделения РЦБЗ взаимодействует с техникой других подразделений при:
- Охране объектов с ФГП "ВО ЖДТ России"
- Координации с ЦЧС (Ситуационный центр мониторинга ЧС)
- Работе с МВД
- Взаимодействии с БВС (Беспилотное воздушное судно)
