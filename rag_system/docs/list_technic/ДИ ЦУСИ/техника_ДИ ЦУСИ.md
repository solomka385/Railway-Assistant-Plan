# Техника ДИ ЦУСИ — Центр управления содержанием инфраструктуры

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДИ ЦУСИ (Центр управления содержанием инфраструктуры) для мониторинга состояния инфраструктуры, диагностики и предотвращения аварий.

## Назначение

Мониторинг состояния инфраструктуры, диагностика, предотвращение аварий.

## Технические средства

### Датчики смещения рельсов DS-200
- Серия: DS-200
- Точность: 0.1 мм
- Тип передачи данных: GSM/GPRS
- Назначение: стационарный мониторинг положения рельсов
- Автоматическая передача данных в центр управления

### Георадары GPR Live
- Назначение: контроль основания пути
- Глубина сканирования: до 5 м
- Мобильность: переносные и на базе транспортных средств
- Обработка данных в реальном времени

### Лазерные нивелиры Leica LS15
- Назначение: измерение геометрии пути
- Точность: ±0.5 мм
- Автоматическое выравнивание
- Интеграция с системами анализа данных

### Мобильные диагностические комплексы
- База: УАЗ "Патриот"
- Оборудование:
  - Ультразвуковой дефектоскоп УЗД-77
  - Георадар GPR Live
  - Лазерный нивелир Leica LS15
  - Портативный спектрометр для анализа металла
- Назначение: оперативный анализ состояния пути, балласта, искусственных сооружений

### Программное обеспечение "ИнфраСкан"
- Назначение: анализ данных, прогнозирование износа
- Функции:
  - Обработка данных с датчиков
  - Прогнозирование дефектов
  - Генерация отчётов
  - Интеграция с системами управления

## Примеры применения

**7 апреля 2024, перегон "Подольск — Климовск"**
Система зафиксировала смещение рельсов. Остановка движения, ремонт за 6 часов — авария предотвращена.

Датчики DS-200 зафиксировали смещение рельсов на 8 мм, что превысило допустимые значения. Система автоматически передала данные в центр управления, было принято решение об остановке движения и вызове ремонтной бригады. Благодаря своевременному обнаружению дефекта удалось предотвратить аварию.

## Взаимодействие с другими подразделениями

Техника подразделения ДИ ЦУСИ взаимодействует с техникой других подразделений при:
- Плановых ремонтах с ДРП (Московская дирекция по ремонту пути)
- Диагностике с ЦТО (Производственный участок мониторинга и диагностики)
- Координации работ с ДЦС (Центр организации работы станций)
- Аварийных ситуациях с ЦЧС (Ситуационный центр мониторинга ЧС)
