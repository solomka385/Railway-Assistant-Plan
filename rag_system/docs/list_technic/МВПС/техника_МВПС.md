# Техника МВПС — моторвагонный подвижной состав

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением МВПС (Моторвагонный подвижной состав) для перевозки пассажиров на пригородных и межрегиональных маршрутах.

## Назначение

Перевозка пассажиров на пригородных и межрегиональных маршрутах.

## Технические средства

### Электропоезда ЭД4М
- Количество вагонов: 10
- Максимальная скорость: до 130 км/ч
- Характеристики:
  - Электропитание: контактная сеть 3 кВ постоянного тока
  - Система тягового привода: асинхронные двигатели
  - Система торможения: пневматическая и рекуперативная
- Оснащение:
  - Системы диагностики на борту
  - Аварийные выходы
  - Огнетушители
  - Аптечки

### Электрички "Ласточка"
- Количество вагонов: 5
- Максимальная скорость: до 160 км/ч
- Характеристики:
  - Электропитание: контактная сеть 3 кВ постоянного тока
  - Система тягового привода: асинхронные двигатели
  - Система торможения: пневматическая и рекуперативная
- Оснащение:
  - Климат-контроль
  - Wi-Fi
  - Розетки для пассажиров
  - Системы диагностики на борту
  - Аварийные выходы
  - Огнетушители
  - Аптечки

### Аэроэкспрессы
- Характеристики:
  - Комфортные вагоны
  - Wi-Fi
  - Розетки
  - Климат-контроль
  - Системы диагностики на борту
- Оснащение:
  - Аварийные выходы
  - Огнетушители
  - Аптечки

### Системы диагностики на борту
- Контроль тормозов
- Контроль контактных токоприёмников
- Контроль систем безопасности
- Контроль систем жизнеобеспечения

### Аварийное оборудование
- Аварийные выходы
- Огнетушители
- Аптечки
- Средства связи

## Технические характеристики

### Регламент эксплуатации электропоездов

**Ежедневно:**
- Визуальный осмотр подвижного состава
- Проверка систем торможения
- Проверка систем безопасности
- Проверка аварийного оборудования

**Перед рейсом:**
- Проверка систем диагностики
- Проверка контактных токоприёмников
- Проверка систем жизнеобеспечения
- Проверка аварийного оборудования

**После рейса:**
- Визуальный осмотр
- Проверка систем диагностики
- Запись в журнал

**Техническое обслуживание:**
- ТО-1 — ежедневно
- ТО-2 — каждые 72 часа
- ТО-3 — каждые 30 дней
- Капитальный ремонт — по износу

### Регламент эксплуатации систем диагностики

**Ежедневно:**
- Проверка работоспособности систем
- Анализ данных диагностики
- Запись в журнал

**Еженедельно:**
- Полная диагностика всех систем
- Анализ статистики отказов
- Настройка параметров

**Ежемесячно:**
- Обновление программного обеспечения
- Калибровка датчиков
- Проверка систем хранения данных

## Регламент эксплуатации и обслуживания

### Подвижной состав
- Ежедневный визуальный осмотр
- Плановое техническое обслуживание по графику
- Текущий ремонт — по необходимости
- Капитальный ремонт — по износу

### Системы диагностики
- Ежедневная проверка работоспособности
- Еженедельная полная диагностика
- Ежемесячное обновление ПО
- Техническое обслуживание — ежемесячно

### Аварийное оборудование
- Ежедневная проверка наличия и работоспособности
- Еженедельная проверка сроков годности
- Замена — по истечении сроков годности

## Примеры применения

**10 мая 2024, перегон "Наро-Фоминск — Бекасово"**
Отказ тормозов. Экстренная остановка, сход 2 вагонов. Эвакуация за 2 часа.

При отказе тормозов:
- Система диагностики зафиксировала отказ
- Была выполнена экстренная остановка
- Произошёл сход 2 вагонов
- Организована эвакуация пассажиров
- Вызваны аварийные службы

## Взаимодействие с другими подразделениями

Техника подразделения МВПС взаимодействует с техникой других подразделений при:
- Техническом обслуживании с ЦТО (Производственный участок мониторинга и диагностики)
- Диагностике с ДИ (Московская дирекция инфраструктуры)
- Ремонте с ДРП (Московская дирекция по ремонту пути)
- Медицинском обслуживании с МАБ
