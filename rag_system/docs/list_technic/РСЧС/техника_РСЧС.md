# Техника РСЧС — Единая государственная система по ЧС

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением РСЧС (Единая государственная система по ЧС) для федеральной координации при крупных ЧС.

## Назначение

Федеральная координация при крупных ЧС.

## Технические средства

### Спутниковые системы мониторинга
- Характеристики:
  - Данные от КА "Канопус", "Электро"
  - Обновление данных в реальном времени
  - Высокое разрешение
- Назначение: мониторинг территории с космоса

### Центр управления в кризисных ситуациях
- Расположение: Москва
- Оснащение:
  - Видеостена
  - АРМ операторов
  - Системы связи
  - Системы анализа данных

### Резервные фонды техники и материалов
- Расположение: по всей стране
- Содержание:
  - Специальная техника
  - Материалы для ликвидации ЧС
  - Оборудование для аварийных работ

### Системы оповещения населения
- Каналы оповещения:
  - SMS-оповещения
  - Сирены
  - ТВ и радио
  - Мобильные приложения
  - Социальные сети
- Назначение: оповещение населения о ЧС

### Вертолёты и самолёты
- Характеристики:
  - Вертолёты Ми-8, Ка-32
  - Самолёты Ил-76, Ан-12
- Назначение: доставка ресурсов и эвакуация

### Специальное оборудование
- Системы видеоконференцсвязи
- Системы документирования
- Средства связи
- Оборудование для работы с документами

## Технические характеристики

### Регламент эксплуатации спутниковых систем

**Ежедневно:**
- Проверка приёма данных
- Проверка качества изображений
- Проверка систем обработки данных

**Еженедельно:**
- Полная проверка систем
- Анализ качества данных
- Проверка систем хранения

**Ежемесячно:**
- Обновление программного обеспечения
- Калибровка систем
- Проверка систем резервирования

### Регламент эксплуатации центра управления

**Ежедневно:**
- Проверка работоспособности видеостены
- Проверка АРМ операторов
- Проверка систем связи
- Проверка систем анализа данных

**Еженедельно:**
- Полная диагностика систем
- Тестирование систем резервирования
- Проверка систем хранения данных

**Ежемесячно:**
- Обновление программного обеспечения
- Техническое обслуживание оборудования
- Проверка систем охлаждения

### Регламент эксплуатации систем оповещения

**Ежедневно:**
- Проверка работоспособности всех каналов
- Тестирование систем оповещения

**Еженедельно:**
- Полное тестирование всех каналов
- Проверка систем резервирования

**Ежемесячно:**
- Обновление баз данных
- Техническое обслуживание оборудования

## Регламент эксплуатации и обслуживания

### Спутниковые системы
- Ежедневная проверка работоспособности
- Еженедельная полная проверка
- Ежемесячное обновление ПО
- Техническое обслуживание — ежемесячно

### Центр управления
- Ежедневная проверка работоспособности
- Еженедельная диагностика
- Ежемесячное ТО
- Капитальный ремонт — по износу

### Системы оповещения
- Ежедневная проверка работоспособности
- Еженедельное тестирование
- Ежемесячное ТО
- Обновление баз данных — по необходимости

### Авиационная техника
- Ежедневный визуальный осмотр
- Плановое ТО — по налёту часов
- Текущий ремонт — по необходимости
- Капитальный ремонт — по налёту

## Примеры применения

**Землетрясение в Байкальском регионе**
РСЧС направила ресурсы — восстановление за 5 дней.

При землетрясении были задействованы:
- Спутниковые системы для мониторинга ситуации
- Центр управления для координации работ
- Вертолёты и самолёты для доставки ресурсов
- Системы оповещения для информирования населения

## Взаимодействие с другими подразделениями

Техника подразделения РСЧС взаимодействует с техникой других подразделений при:
- Координации со всеми силами — МЧС, РЖД, Минобороны, ФСБ
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
- Аварийных ситуациях с ДАВС (Дирекция аварийно-восстановительных средств)
- Взаимодействии с региональными комиссиями по ЧС

4. Правила работы с авиационной техникой
5. Инструкция по действиям при чрезвычайных ситуациях
6. Правила работы с документами
