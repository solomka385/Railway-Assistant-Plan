# Техника ДАВС — Дирекция аварийно-восстановительных средств

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДАВС (Дирекция аварийно-восстановительных средств) для координации всех сил, обеспечения техникой, логистики и стратегического управления при чрезвычайных ситуациях.

## Назначение

Координация всех сил, обеспечение техникой, логистика, стратегическое управление при ЧС.

## Технические средства

### Аварийно-восстановительные поезда (АВП)
- Количество: 47 шт., размещены по России
- Модель: АВП-125М
- Состав: 12 вагонов
  - Кран-балка грузоподъёмностью 125 т, гидравлический привод, радиоуправление, радиус стрелы 18 м
  - Платформы для перевозки рельсовых секций (до 25 м), шпал, блоков
  - Вагон-лаборатория с диагностическим оборудованием (дефектоскопы, тепловизоры)
  - Вагон-бытовка на 15 человек, оборудован кухней, душевой
  - Вагон-генератор мощностью 300 кВт (дизель)
  - Вагон-мастерская с инверторными сварочными аппаратами FUBAG 250, компрессором
  - Вагон-склад запасных частей (рельсовые накладки, болты, изоляторы)
  - Вагон-водовоз на 10 т воды
  - Вагон-пожарный модуль с насосом производительностью 40 л/с, пожарными рукавами
  - Вагон-электроподстанция 10/0,4 кВ, мощностью 400 кВА
  - Вагон-связи со спутниковым терминалом Inmarsat, УКВ-радиостанцией
  - Вагон-медицинский пункт с дефибриллятором, ИВЛ, носилками

### Мобильные командные пункты
- Шасси: КАМАЗ-6350
- Оснащение:
  - Системы связи (спутниковая, радиорелейная)
  - Видеостена
  - Серверы
  - Рабочие места операторов

### Резервные склады
- Количество: 3 шт. (Москва, Екатеринбург, Красноярск)
- Содержание: запчасти, материалы, оборудование

### Авиагруппы
- Вертолёты: Ми-8МТВ-2 — 5 шт.
- Назначение: доставка бригад в труднодоступные места

### Пожарные машины
- Модель: АЦ-40 на КАМАЗ-5350
- Количество: 20 шт.
- Характеристики:
  - Цистерна 4 т воды
  - Насос 40 л/с
  - Система подачи пены
  - Генератор 10 кВт

## Технические характеристики

### Регламент обслуживания АВП-125М

**Ежедневно:**
- Проверка гидравлических магистралей
- Проверка давления в системах
- Проверка тормозов
- Проверка зарядки аккумуляторов

**Еженедельно:**
- Испытание крана на полную грузоподъёмность
- Тестирование сварочного оборудования
- Тестирование систем связи

**Ежемесячно:**
- Полная диагностика комплексом «РЖД-Диагноз»

**Каждые 6 месяцев:**
- Замена гидравлической жидкости
- Профилактический ремонт ходовых частей

**После каждой аварийной операции:**
- Осмотр всех механизмов
- Запись в журнал неисправностей

**Зимой:**
- Предпусковой прогрев гидравлики
- Антиобледенительная обработка стрелы

**Норматив готовности:** выезд в течение 15 мин после сигнала

### Регламент эксплуатации пожарных машин АЦ-40

**Перед выездом:**
- Проверка уровня воды
- Проверка давления
- Проверка насоса
- Проверка заряда аккумуляторов

**При t<-35°C обязателен антифриз**

**При тушении электроустановок — только пена**

## Примеры применения

**3 августа 2023, Краснодарский край, мост через Кубань**
Использован АВП №7, авиагруппа, мобильный КП. Восстановление за 72 часа.

При аварии был задействован полный комплекс аварийно-восстановительных средств: поезд АВП-125М для восстановления мостовой конструкции, вертолёты Ми-8 для доставки бригад и эвакуации пострадавших, мобильный командный пункт для координации работ.
