# Техника НУЗ — Научно-учебный центр

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением НУЗ (Научно-учебный центр) для обучения персонала и проведения исследований.

## Назначение

Обучение персонала, проведение исследований, разработка методик.

## Технические средства

### Учебные классы
- Оснащение:
  - Компьютеры с обучающим ПО
  - Проекторы и экраны
  - Интерактивные доски
  - Системы видеоконференцсвязи
- Назначение: теоретическое обучение персонала

### Тренажёры
- Типы:
  - Тренажёры диспетчеров
  - Тренажёры машинистов
  - Тренажёры аварийно-восстановительных работ
- Характеристики:
  - Реалистичное моделирование ситуаций
  - Система оценки действий
  - Возможность записи и анализа

### Лабораторное оборудование
- Оборудование:
  - Приборы для исследований
  - Испытательные стенды
  - Измерительные приборы
  - Компьютеры для обработки данных
- Назначение: проведение исследований и испытаний

### Библиотека и информационный центр
- Оснащение:
  - Электронная библиотека
  - Доступ к базам данных
  - Компьютеры для работы
  - Система поиска информации

### Мультимедийное оборудование
- Оснащение:
  - Видеосистема
  - Аудиосистема
  - Система записи лекций
  - Система дистанционного обучения

## Примеры применения

**Обучение диспетчеров**
Проведено обучение 50 диспетчеров на тренажёрах с использованием реальных сценариев аварийных ситуаций.

При обучении были задействованы:
- Тренажёры диспетчеров для моделирования ситуаций
- Учебные классы для теоретического обучения
- Мультимедийное оборудование для проведения лекций
- Система видеоконференцсвязи для дистанционного обучения

## Взаимодействие с другими подразделениями

Техника подразделения НУЗ взаимодействует с техникой других подразделений при:
- Обучении персонала всех подразделений
- Проведении исследований для ДИ, ДРП, ЦТО
- Разработке методик для ЦЧС, ДАВС
- Координации с учебными центрами других дирекций
