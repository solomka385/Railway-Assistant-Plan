# Техника СЦ — Ситуационный центр Московской ж/д

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением СЦ (Ситуационный центр Московской ж/д) для мониторинга и принятия решений при отказах техники.

## Назначение

Мониторинг и принятие решений при отказах техники.

## Технические средства

### Видеостена
- Размер: 3x3 м
- Назначение: отображение всех систем
- Характеристики:
  - Высокое разрешение
  - Множество источников сигнала
  - Система управления

### АРМ диспетчера
- Характеристики:
  - Доступ ко всем АСУ
  - Многомониторные рабочие станции
  - Резервное питание
- Назначение: работа с системами управления

### Системы Big Data
- Характеристики:
  - Анализ отказов
  - Прогнозирование ситуаций
  - Обработка больших объёмов данных
- Назначение: анализ данных и принятие решений

### Мобильные приложения
- Характеристики:
  - Передача задач бригадам
  - Получение отчётов
  - Геолокация
- Назначение: связь с полевыми бригадами

### Резервные каналы связи
- Характеристики:
  - Спутниковые каналы
  - Радиорелейные линии
  - Оптоволокно с резервированием
- Назначение: обеспечение связи при отказах

### Специальное оборудование
- Системы видеоконференцсвязи
- Системы документирования
- Средства связи
- Оборудование для работы с документами

## Примеры применения

**Массовый отказ светофоров**
СЦ локализовал проблему за 10 минут — движение восстановлено.

При массовом отказе светофоров:
- Системы Big Data проанализировали ситуацию
- Видеостена отобразила все отказы
- АРМ диспетчера позволили принять решение
- Мобильные приложения передали задачи бригадам

## Взаимодействие с другими подразделениями

Техника подразделения СЦ взаимодействует с техникой других подразделений при:
- Координации с ДЦС (Центр организации работы станций)
- Управлении с ДЦУП (Диспетчерский центр управления перевозками)
- Техническом обслуживании с ЦТО (Производственный участок мониторинга и диагностики)
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
