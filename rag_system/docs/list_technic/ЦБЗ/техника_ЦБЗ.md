# Техника ЦБЗ — Центр безопасности

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦБЗ (Центр безопасности) для обеспечения безопасности объектов и контроля доступа.

## Назначение

Обеспечение безопасности объектов, контроль доступа.

## Технические средства

### Системы видеонаблюдения
- Характеристики:
  - IP-камеры с высоким разрешением
  - ИИ-аналитика
  - Запись видео
  - Ночное видение
- Назначение: наблюдение за объектами

### Системы контроля доступа
- Характеристики:
  - RFID-карты
  - Биометрическая идентификация
  - Автоматические турникеты
- Назначение: контроль доступа на объекты

### Системы сигнализации
- Характеристики:
  - Датчики движения
  - Датчики открытия
  - Система оповещения
- Назначение: защита от несанкционированного доступа

### Средства связи
- Характеристики:
  - Радиостанции
  - Телефоны
  - Системы видеоконференцсвязи
- Назначение: связь между постами охраны

### Специальное оборудование
- Металлодетекторы
- Рентгеновские сканеры
- Системы досмотра
- Средства индивидуальной защиты

## Примеры применения

**Попытка несанкционированного доступа на объект**
Системы видеонаблюдения зафиксировали нарушение, охрана предотвратила проникновение.

При попытке несанкционированного доступа были задействованы:
- Системы видеонаблюдения для фиксации нарушения
- Системы контроля доступа для предотвращения проникновения
- Системы сигнализации для оповещения охраны
- Средства связи для координации действий

## Взаимодействие с другими подразделениями

Техника подразделения ЦБЗ взаимодействует с техникой других подразделений при:
- Охране объектов с ФГП «ВО ЖДТ России»
- Координации с РЦБЗ (Московский региональный центр безопасности)
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
- Взаимодействии с МВД
