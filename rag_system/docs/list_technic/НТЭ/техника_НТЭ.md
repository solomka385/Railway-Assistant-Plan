# Техника НТЭ — Московская дирекция по энергообеспечению

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением НТЭ (Московская дирекция по энергообеспечению) для обеспечения электроснабжения объектов инфраструктуры.

## Назначение

Обеспечение электроснабжения объектов инфраструктуры.

## Технические средства

### Передвижные подстанции на КАМАЗ-6350
- Характеристики:
  - Мощность: 1600 кВА
  - Дизель-генератор: Cummins QSK19 (1200 кВт)
  - Трансформатор: 10/0,4 кВ
- Оснащение:
  - Система автоматического включения резерва (АВР)
  - Защита от перегрузки и КЗ
  - Кабели и соединительные устройства

### Дизель-генераторы SDMO T16
- Характеристики:
  - Мощность: 1500 кВт
  - Автозапуск при пропадании сети
  - Резервное топливо на 24 часа
- Назначение: резервное питание объектов инфраструктуры

### Кабелеукладчики
- Характеристики:
  - Глубина прокладки: до 1.5 м
  - Ширина траншеи: до 0.5 м
- Назначение: прокладка резервных линий

### Тепловизоры Testo 885
- Характеристики:
  - Разрешение: 640×480
  - Температурный диапазон: -30°C до +650°C
- Назначение: контроль нагрева оборудования

### Мобильные лаборатории
- Оборудование:
  - Приборы для испытания изоляции
  - Измерительные приборы
  - Оборудование для анализа качества электроэнергии
- Назначение: испытание и диагностика оборудования

### Специальное оборудование
- Комплекты кабелей и соединителей
- Системы заземления
- Защитное оборудование
- Средства измерения

## Технические характеристики

### Регламент эксплуатации передвижных подстанций

**Перед подключением:**
- Проверка заземления
- Проверка напряжения на входе
- Проверка состояния кабелей
- Проверка систем защиты

**Во время работы:**
- Постоянный мониторинг температуры трансформатора
- Контроль уровня масла
- Контроль за нагрузкой
- Проверка систем охлаждения

**После использования:**
- Отключение нагрузки
- Проверка состояния оборудования
- Запись в журнал

**Готовность к выезду: 10 мин**

**Запрещено подключение при повреждённой изоляции**

**Работа только с двумя операторами**

### Регламент эксплуатации дизель-генераторов

**Еженедельно:**
- Тестовый запуск
- Проверка уровня топлива
- Проверка уровня масла
- Проверка систем охлаждения

**При работе:**
- Контроль за нагрузкой
- Контроль за температурой
- Проверка систем защиты

**После использования:**
- Проверка состояния оборудования
- Запись в журнал

**Техническое обслуживание:**
- ТО — каждые 250 моточасов
- Капитальный ремонт — по износу

### Регламент эксплуатации тепловизоров

**Перед использованием:**
- Проверка заряда аккумуляторов
- Проверка работоспособности
- Калибровка

**Во время использования:**
- Контроль за качеством изображения
- Проверка температурного диапазона

**После использования:**
- Очистка объектива
- Проверка на наличие повреждений
- Зарядка аккумуляторов

## Примеры применения

**Ураган в Москве**
Отключение 5 станций. НТЭ восстановила питание за 2 часа через резерв.

При урагане были задействованы:
- Передвижные подстанции для восстановления питания
- Дизель-генераторы для резервного питания
- Кабелеукладчики для прокладки резервных линий
- Тепловизоры для контроля состояния оборудования

## Взаимодействие с другими подразделениями

Техника подразделения НТЭ взаимодействует с техникой других подразделений при:
- Обеспечении связи с ЦСС (Центральная станция связи)
- Управлении с ЦТУ (Отдел технического управления сетями связи)
- Ремонте с ДИ (Московская дирекция инфраструктуры)
- Координации с ДЦС (Центр организации работы станций)
