# Техника ДЦС — центр организации работы станций

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДЦС (Центр организации работы станций) для координации работы станций, манёвров и формирования составов.

## Назначение

Координация работы станций, маневры, формирование составов.

## Технические средства

### АРМ диспетчера станции
- Количество мониторов: 3
- Программное обеспечение:
  - "Станция-2025" — система управления станцией
  - Модуль маневровой работы
  - Модуль формирования составов
- Резервное питание: ИБП на 8 часов

### Системы видеоаналитики Hikvision
- Характеристики:
  - ИИ-распознание
  - Распознавание номеров вагонов
  - Контроль за безопасностью
- Интеграция с АРМ диспетчера

### Электронные табло на платформах
- Характеристики:
  - LED-дисплеи
  - Управление через центр
  - Автоматическое обновление информации
- Отображение расписания и задержек

### Радиостанции
- Модель: Motorola DP4801
- Диапазон: УКВ
- Стандарт: цифровой DMR
- Назначение: связь с бригадами

### Система контроля доступа на перроны
- Оборудование:
  - Турникеты
  - Сканеры билетов
  - Системы видеонаблюдения
- Интеграция с системами безопасности

### Резервное оборудование
- Источник бесперебойного питания (ИБП) на 8 часов
- Резервные каналы связи
- Дублирующие рабочие места

## Технические характеристики

### Регламент эксплуатации АРМ диспетчера станции

**Ежедневно:**
- Проверка работоспособности всех мониторов
- Проверка запуска программного обеспечения
- Проверка связи с системами управления
- Проверка резервного питания

**Еженедельно:**
- Тестирование систем видеоаналитики
- Проверка электронных табло
- Проверка систем контроля доступа

**Ежемесячно:**
- Полная диагностика оборудования
- Обновление программного обеспечения
- Проверка систем резервирования

### Регламент эксплуатации систем видеоаналитики

**Ежедневно:**
- Проверка записи видео
- Проверка работы ИИ-распознавания
- Проверка связи с АРМ диспетчера

**Еженедельно:**
- Очистка объективов камер
- Проверка настроек распознавания
- Тестирование системы оповещения

**Ежемесячно:**
- Обновление прошивки камер
- Проверка систем хранения данных

### Регламент эксплуатации радиостанций

**Ежедневно:**
- Проверка заряда аккумуляторов
- Проверка чистки контактов
- Тестирование связи

**Еженедельно:**
- Полная проверка работоспособности
- Проверка антенн

- Ежедневная проверка работоспособности

- Техническое обслуживание — ежемесячно

## Примеры применения

**22 декабря 2023, Курский вокзал**
Отключение электропитания. ДЦС включил резерв, перенаправил поезда, организовал автобусы — без отмен.

При отключении электропитания диспетчер:
- Включил резервное питание
- Перенаправил поезда по альтернативным маршрутам
- Организовал автобусное сообщение для пассажиров
- Вёл информирование пассажиров через электронные табло

## Взаимодействие с другими подразделениями

Техника подразделения ДЦС взаимодействует с техникой других подразделений при:
- Оперативном управлении с ДЦУП (Диспетчерский центр управления перевозками)
- Координации с ЦЧС (Ситуационный центр мониторинга ЧС)
- Работе с МАБ (Медицинская выездная бригада)
- Охране объектов с ФГП
- Аварийных ситуациях с МЧС
