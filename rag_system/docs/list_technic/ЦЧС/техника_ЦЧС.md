# Техника ЦЧС — Ситуационный центр мониторинга ЧС

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦЧС (Ситуационный центр мониторинга ЧС) для мониторинга и управления при ЧС.

## Назначение

Мониторинг и управление при ЧС.

## Технические средства

### Видеостена
- Размер: 6x3 м
- Характеристики:
  - Карта ЧС в реальном времени
  - Высокое разрешение
  - Множество источников сигнала
- Назначение: отображение ситуации при ЧС

### Системы Big Data
- Характеристики:
  - Анализ данных
  - Прогнозирование ситуаций
  - Обработка больших объёмов данных
- Назначение: анализ данных и принятие решений

### Спутниковые снимки
- Характеристики:
  - От КА "Канопус"
  - Высокое разрешение
  - Обновление в реальном времени
- Назначение: мониторинг территории

### Датчики
- Типы:
  - Сейсмодатчики
  - Гидродатчики
  - Термодатчики
- Характеристики:
  - Автоматическая передача данных
  - Высокая точность
- Назначение: мониторинг параметров среды

### Системы оповещения
- Каналы оповещения:
  - SMS
  - Сирены
  - Приложения
  - ТВ и радио
- Назначение: оповещение о ЧС

### Специальное оборудование
- Системы видеоконференцсвязи
- Системы документирования
- Средства связи
- Оборудование для работы с документами

## Примеры применения

**Разлив нефти**
ЦЧС смоделировал распространение, направил бригады — ликвидация за 6 часов.

При разливе нефти были задействованы:
- Видеостена для отображения ситуации
- Системы Big Data для моделирования распространения
- Спутниковые снимки для мониторинга территории
- Датчики для контроля параметров среды
- Системы оповещения для информирования населения

## Взаимодействие с другими подразделениями

Техника подразделения ЦЧС взаимодействует с техникой других подразделений при:
- Координации со всеми подразделениями
- Аварийных ситуациях с МЧС
- Работе с РСЧС (Единая государственная система по ЧС)
- Взаимодействии с ДАВС (Дирекция аварийно-восстановительных средств)
