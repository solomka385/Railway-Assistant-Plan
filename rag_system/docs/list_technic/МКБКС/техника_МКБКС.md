# Техника МКБКС — мобильный комплекс видео-конференц-связи

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением МКБКС (Мобильный комплекс видео-конференц-связи) для обеспечения связи между полевыми бригадами и центральным штабом при ЧС.

## Назначение

Обеспечение связи между полевыми бригадами и центральным штабом при ЧС.

## Технические средства

### Комплекс на шасси КАМАЗ-6350
- Характеристики:
  - Полный привод
  - Высокая проходимость
- Оснащение:
  - Антенна спутниковой связи
  - Видеоконференц-система
  - Серверы
  - Дизель-генератор
  - Система шифрования

### Спутниковая антенна Inmarsat BGAN
- Характеристики:
  - Скорость передачи данных: до 492 кбит/с
  - Автоматическое наведение
  - Защита от погодных условий
- Назначение: обеспечение связи в удалённых районах

### Видеоконференц-система Cisco Webex Room Kit
- Характеристики:
  - 4K камера
  - 8 микрофонов
  - 16" экран
  - Система шумоподавления
- Функции:
  - Видеоконференции
  - Обмен документами
  - Запись встреч

### Серверы HP ProLiant DL380
- Характеристики:
  - 32 ГБ ОЗУ
  - Высокая производительность
  - Система резервирования
- Назначение: хранение и передача данных

### Дизель-генератор SDMO T16
- Характеристики:
  - Мощность: 30 кВт
  - Автономность: 72 часа
  - Автоматический запуск
- Назначение: автономное электроснабжение

### Система шифрования AES-256
- Назначение: защита передаваемых данных
- Характеристики:
  - Стандарт шифрования: AES-256
  - Автоматическое шифрование
  - Защита от перехвата

### Защищённые планшеты Getac
- Характеристики:
  - Защита от пыли и влаги
  - Ударопрочность
  - Длительная автономность
- Назначение: передача видео в реальном времени

## Технические характеристики

### Регламент эксплуатации комплекса

**Перед развёртыванием:**
- Проверка технического состояния шасси
- Проверка работоспособности спутниковой антенны
- Проверка видеоконференц-системы
- Проверка серверов
- Проверка дизель-генератора
- Проверка системы шифрования

**Развёртывание:**
- Время развёртывания: 15 мин после прибытия
- Требуемая площадка: 6×6 м
- Проверка связи со спутником
- Проверка видеоконференц-связи

**Во время работы:**
- Контроль за качеством связи
- Контроль за работой оборудования
- Контроль за уровнем топлива

**После использования:**
- Очистка антенны
- Проверка герметичности
- Зарядка резервных аккумуляторов
- Запись в журнал

**Запрещена работа:**
- Вблизи высоковольтных линий
- При t<-40°C без прогрева

### Регламент эксплуатации спутниковой антенны

**Перед использованием:**
- Проверка работоспособности
- Проверка наведения на спутник
- Проверка качества сигнала

**Во время использования:**
- Контроль за качеством сигнала
- Контроль за состоянием антенны

**После использования:**
- Очистка антенны
- Проверка на наличие повреждений
- Запись в журнал

### Регламент эксплуатации видеоконференц-системы

**Перед использованием:**
- Проверка работоспособности камеры
- Проверка микрофонов
- Проверка экрана
- Проверка связи

**Во время использования:**
- Контроль за качеством видео и звука
- Контроль за записью

**После использования:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

## Взаимодействие с другими подразделениями

Техника подразделения МКБКС взаимодействует с техникой других подразделений при:
- Координации с ЦЧС (Ситуационный центр мониторинга ЧС)
- Управлении с ЦУП (Центр управления перевозками Центральной дирекции)
- Аварийных ситуациях с МЧС
- Взаимодействии с ЦД (Центральная дирекция управления движением)
- Работе с РСЧС (Единая государственная система по ЧС)
