# Техника ЦТО — производственный участок мониторинга и диагностики

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦТО (Производственный участок мониторинга и диагностики) для диагностики технического состояния подвижного состава.

## Назначение

Диагностика технического состояния подвижного состава.

## Технические средства

### Диагностические стенды «Диагност-2025»
- База: ГАЗель NEXT
- Состав:
  - Сканер OBD-2 с поддержкой протоколов РЖД-1/РЖД-2
  - Осциллограф
  - Термокамера
  - Датчики давления в тормозных цилиндрах
  - Анализатор качества электроэнергии
  - ПО с базой данных локомотивов и вагонов
- Назначение: мобильная диагностика тормозных систем, электроники, тяговых двигателей

### Мобильные лаборатории
- База: ГАЗель NEXT
- Оборудование:
  - Диагностические приборы
  - Средства измерения
  - Оборудование для анализа данных
- Назначение: оперативная диагностика

### Сканеры RFID
- Характеристики:
  - Учёт неисправностей
  - Автоматическое считывание данных
  - Интеграция с базой данных
- Назначение: учёт и контроль неисправностей

### Планшеты с доступом к базе данных
- Характеристики:
  - Защищённые планшеты
  - Доступ к базе данных
  - Автоматическое обновление данных
- Назначение: работа с данными о подвижном составе

### Специальное оборудование
- Приборы для измерения
- Оборудование для анализа
- Средства связи
- Оборудование для работы с документами

## Технические характеристики

### Регламент эксплуатации диагностических стендов

**Перед диагностикой:**
- Подключение к штатному разъёму
- Проверка напряжения бортовой сети
- Проверка работоспособности оборудования

**Запрещена диагностика при включённом двигателе без защиты от перенапряжения**

**Во время диагностики:**
- Контроль за качеством измерений
- Проверка систем безопасности
- Запись данных

**После диагностики:**
- Формирование отчёта XML
- Передача в ЦД и ДИ ЦУСИ
- Запись в журнал

**Техническое обслуживание:**
- Плановое ТО — ежемесячно
- Калибровка — ежеквартально
- Поверка — ежегодно

### Регламент эксплуатации мобильных лабораторий

**Перед выездом:**
- Проверка технического состояния автомобиля
- Проверка работоспособности оборудования
- Проверка зарядки аккумуляторов

**Во время работы:**
- Диагностика подвижного состава
- Запись данных
- Анализ результатов

**После работы:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

## Примеры применения

**Диагностика локомотива**
Выявлен износ тормозных колодок, предотвращён выпуск в рейс.

При диагностике локомотива были задействованы:
- Диагностические стенды «Диагност-2025» для проверки систем
- Мобильные лаборатории для оперативной диагностики
- Сканеры RFID для учёта неисправностей
- Планшеты для работы с базой данных

## Взаимодействие с другими подразделениями

Техника подразделения ЦТО взаимодействует с техникой других подразделений при:
- Диагностике с ДИ (Московская дирекция инфраструктуры)
- Управлении с ЦТУ (Отдел технического управления сетями связи)
- Работе с дирекциями подвижного состава
- Координации с ЦЧС (Ситуационный центр мониторинга ЧС)
