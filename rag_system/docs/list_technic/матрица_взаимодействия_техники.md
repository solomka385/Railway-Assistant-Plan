## 2. Сценарии совместной работы техники

### 2.1 Сценарий: Подъём и перемещение груза

#### Задействованная техника
- Кран КС-5371
- Стропы и грузозахватные приспособления
- Рация Motorola GP-340

#### Последовательность действий

1. **Подготовка**
   - Кран устанавливается на позицию
   - Стропы подготавливаются к работе
   - Связь проверяется

2. **Строповка**
   - Стропальщик выполняет строповку груза
   - Проверяется правильность строповки
   - Подаётся сигнал машинисту

3. **Подъём**
   - Машинист получает сигнал
   - Груз плавно поднимается
   - Стропальщик контролирует подъём

4. **Перемещение**
   - Груз перемещается к месту разгрузки
   - Стропальщик контролирует направление
   - Груз опускается

5. **Отстроповка**
   - Груз отстроповывается
   - Кран освобождается

### 2.2 Сценарий: Расчистка завала

#### Задействованная техника
- Экскаватор Э-153
- Бульдозер Т-170
- Погрузчик Амкодор-332
- КАМАЗ-5320

#### Последовательность действий

1. **Оценка завала**
   - Экскаватор оценивает объём завала
   - Определяется порядок работ

2. **Расчистка**
   - Экскаватор разрабатывает завал
   - Бульдозер планирует территорию
   - Погрузчик загружает обломки

3. **Транспортировка**
   - КАМАЗ транспортирует обломки
   - Погрузчик загружает КАМАЗ
   - Обломки вывозятся

4. **Завершение**
   - Территория очищается
   - Техника освобождается

### 2.3 Сценарий: Укладка пути

#### Задействованная техника
- Путеукладчик УК-25
- Щебнеочиститель ЩОМ-4
- ВПР-02
- Балластировщик Б-М

#### Последовательность действий

1. **Подготовка**
   - Щебнеочиститель очищает балласт
   - Территория подготавливается

2. **Укладка**
   - Путеукладчик укладывает рельсы и шпалы
   - Балластировщик засыпает балласт

3. **Выправка**
   - ВПР-02 выправляет путь
   - Геометрия пути проверяется

4. **Завершение**
   - Путь сдаётся в эксплуатацию
   - Техника освобождается

### 2.4 Сценарий: Восстановление контактной сети

#### Задействованная техника
- Автовышка АГ-60
- Монтажный поезд МП

#### Последовательность действий

1. **Подготовка**
   - Автовышка устанавливается на позицию
   - Монтажный поезд прибывает

2. **Монтаж**
   - Автовышка поднимает монтажников
   - Монтажный поезд подаёт материалы
   - Контактная сеть монтируется

3. **Проверка**
   - Контактная сеть проверяется
   - Напряжение подаётся

4. **Завершение**
   - Движение открывается
   - Техника освобождается

### 2.5 Сценарий: Аэрофотосъёмка

#### Задействованная техника
- Дрон DJI Matrice 30T
- Дрон Autel EVO Max 4T
- Мобильный диагностический комплекс МДК

#### Последовательность действий

1. **Подготовка**
   - Дроны подготавливаются к полёту
   - МДК разворачивается

2. **Полёт**
   - Дроны выполняют съёмку
   - Данные передаются в МДК

3. **Обработка**
   - МДК обрабатывает данные
   - Анализ выполняется

4. **Завершение**
   - Дроны возвращаются
   - Данные сохраняются

## 3. Протоколы совместной работы

### 3.1 Протокол координации крана и стропальщика

| Этап | Кран | Стропальщик | Канал связи |
|------|------|-------------|-------------|
| Подготовка | Установка на позицию | Подготовка стропов | Рация |
| Строповка | Ожидание сигнала | Строповка груза | Рация |
| Подъём | Получение сигнала, подъём | Контроль подъёма | Рация |
| Перемещение | Перемещение груза | Контроль направления | Рация |
| Опускание | Опускание груза | Контроль опускания | Рация |
| Отстроповка | Ожидание сигнала | Отстроповка | Рация |

### 3.2 Протокол координации экскаватора и погрузчика

| Этап | Экскаватор | Погрузчик | Канал связи |
|------|------------|-----------|-------------|
| Подготовка | Оценка завала | Подготовка к погрузке | Рация |
| Разработка | Разработка завала | Ожидание материала | Рация |
| Погрузка | Загрузка ковша | Погрузка материала | Рация |
| Транспортировка | Разработка следующей части | Транспортировка | Рация |

### 3.3 Протокол координации путеукладчика и балластировщика

| Этап | Путеукладчик | Балластировщик | Канал связи |
|------|--------------|----------------|-------------|
| Подготовка | Подготовка к укладке | Подготовка балласта | Рация |
| Укладка | Укладка рельсов | Ожидание | Рация |
| Балластировка | Укладка следующих рельсов | Засыпка балласта | Рация |
| Выправка | Ожидание | Ожидание | Рация |

## 4. Стандарты совместной работы

### 4.1 Стандарты для крановых работ

| Параметр | Значение |
|----------|----------|
| Расстояние между кранами | Не менее 10 м |
| Расстояние до людей | Не менее 5 м |
| Скорость подъёма | Не более 0,5 м/с |
| Скорость перемещения | Не более 1 м/с |
| Частота связи | Постоянная |

### 4.2 Стандарты для землеройных работ

| Параметр | Значение |
|----------|----------|
| Расстояние между экскаваторами | Не менее 5 м |
| Расстояние до людей | Не менее 3 м |
| Глубина копания | Не более 5,5 м |
| Радиус копания | Не более 9,5 м |
| Частота связи | Каждые 5 минут |

### 4.3 Стандарты для погрузочных работ

| Параметр | Значение |
|----------|----------|
| Расстояние между погрузчиками | Не менее 3 м |
| Расстояние до людей | Не менее 2 м |
| Грузоподъёмность | Не более номинальной |
| Скорость перемещения | Не более 10 км/ч |
| Частота связи | Каждые 10 минут |

## 5. Требования безопасности при совместной работе

### 5.1 Общие требования

| Требование | Описание |
|------------|----------|
| Обязательная связь | Все единицы техники должны иметь связь |
| Обязательные СИЗ | Все операторы должны использовать СИЗ |
| Контроль зоны | Зона работы должна быть ограждена |
| Координация действий | Все действия должны быть скоординированы |
| Контроль погоды | Работы прекращаются при неблагоприятной погоде |

### 5.2 Требования для крановых работ

| Требование | Описание |
|------------|----------|
| Запрет работы под стрелой | Запрещено находиться под стрелой |
| Запрет перегруза | Не превышать грузоподъёмность |
| Обязательные стропальщики | Стропальщики обязательны при работе |
| Контроль устойчивости | Контролировать устойчивость крана |
| Контроль ветра | Не работать при ветре более 10 м/с |

### 5.3 Требования для землеройных работ

| Требование | Описание |
|------------|----------|
| Запрет работы под ковшом | Запрещено находиться под ковшом |
| Контроль глубины | Контролировать глубину копания |
| Контроль расстояний | Соблюдать безопасные расстояния |
| Контроль грунта | Учёт типа грунта |
| Контроль откосов | Не работать на крутых откосах |
