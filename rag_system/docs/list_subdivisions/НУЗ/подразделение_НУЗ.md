# НУЗ — Негосударственное учреждение здравоохранения ОАО «РЖД»

## Общее описание

Негосударственное учреждение здравоохранения ОАО «РЖД» (НУЗ) — медицинское учреждение, обеспечивающее медицинское обслуживание работников железнодорожного транспорта и пассажиров. НУЗ оказывает широкий спектр медицинских услуг, включая амбулаторную помощь, стационарное лечение и профилактические мероприятия.

## Основные функции

### Медицинские функции
1. **Амбулаторная помощь** — оказание амбулаторной медицинской помощи
2. **Стационарное лечение** — лечение в стационаре
3. **Скорая помощь** — оказание экстренной медицинской помощи
4. **Профилактические осмотры** — проведение профилактических осмотров

### Диагностические функции
1. **Лабораторная диагностика** — проведение лабораторных исследований
2. **Инструментальная диагностика** — проведение инструментальных исследований
3. **Рентгенодиагностика** — проведение рентгенологических исследований
4. **Ультразвуковая диагностика** — проведение УЗИ исследований

### Организационные функции
1. **Планирование медицинской помощи** — разработка планов оказания помощи
2. **Управление персоналом** — организация работы медицинского персонала
3. **Взаимодействие с другими службами** — координация работы

## Организационная структура

### Руководство
- **Главный врач** — возглавляет учреждение, принимает ключевые решения
- **Заместитель главного врача по лечебной работе** — курирует лечебную деятельность
- **Заместитель главного врача по поликлинике** — курирует поликлиническую службу
- **Главная медсестра** — руководит средним медицинским персоналом

### Поликлиника
- **Заведующий поликлиникой** — руководит работой поликлиники
- **Врачи-специалисты** — оказывают специализированную помощь
- **Процедурные медсестры** — выполняют медицинские процедуры
- **Регистраторы** — ведут прием пациентов

### Стационар
- **Заведующий стационаром** — руководит работой стационара
- **Заведующие отделениями** — руководят отделениями
- **Врачи** — оказывают медицинскую помощь
- **Медсестры** — ухаживают за пациентами

### Диагностическая служба
- **Заведующий лабораторией** — руководит лабораторией
- **Врачи-диагносты** — проводят диагностические исследования
- **Лаборанты** — выполняют лабораторные исследования
