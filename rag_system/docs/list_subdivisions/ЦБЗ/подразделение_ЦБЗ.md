# ЦБЗ — Департамент безопасности ОАО «РЖД»

## Общее описание

Департамент безопасности ОАО «РЖД» (ЦБЗ) — структурное подразделение, обеспечивающее комплексную безопасность деятельности ОАО «РЖД», защиту имущества и персонала. ЦБЗ разрабатывает и реализует политику безопасности, координирует работу служб безопасности и взаимодействует с правоохранительными органами.

## Основные функции

### Руководящие функции
1. **Разработка политики безопасности** — разработка политики безопасности ОАО «РЖД»
2. **Координация работы служб безопасности** — координация работы всех служб безопасности
3. **Контроль соблюдения правил безопасности** — контроль соблюдения правил безопасности

### Оперативные функции
1. **Профилактика нарушений** — профилактика нарушений безопасности
2. **Расследование нарушений** — расследование нарушений безопасности
3. **Взаимодействие с правоохранительными органами** — взаимодействие с МВД, ФСБ, прокуратурой

### Аналитические функции
1. **Анализ угроз** — анализ угроз безопасности
2. **Прогнозирование рисков** — прогнозирование рисков безопасности
3. **Разработка мер по повышению безопасности** — разработка мероприятий

## Организационная структура

### Руководство
- **Директор Департамента** — возглавляет департамент, определяет политику безопасности
- **Заместители директора** — курируют отдельные направления деятельности
- **Начальники управлений** — руководят профильными управлениями

### Управления департамента
- **Управление экономической безопасности** — предотвращает экономические преступления
- **Управление физической защиты** — обеспечивает охрану объектов
- **Управление информационной безопасности** — защищает информационные ресурсы
- **Управление кадровой безопасности** — проводит проверку персонала

### Оперативный персонал
- **Начальник оперативного отдела** — руководит оперативной работой
- **Старшие оперуполномоченные** — проводят оперативные мероприятия
- **Оперуполномоченные** — выполняют оперативные задачи
- **Инспекторы безопасности** — проводят проверки и расследования

### Технический персонал
- **Инженеры по техническим средствам защиты** — обслуживают системы безопасности
- **Администраторы систем безопасности** — управляют программными комплексами

- Принятие решений по реагированию на нарушение

### 3. Проведение оперативных мероприятий

- Ведение документации
