# ЦРБ — Департамент безопасности движения ОАО «РЖД»

## Общее описание

Департамент безопасности движения ОАО «РЖД» (ЦРБ) — структурное подразделение, обеспечивающее безопасность движения поездов на сети железных дорог. ЦРБ разрабатывает и реализует политику безопасности движения, анализирует причины нарушений и разрабатывает меры по их предотвращению.

## Основные функции

### Руководящие функции
1. **Разработка политики безопасности движения** — разработка политики безопасности движения
2. **Координация работы служб безопасности движения** — координация работы всех служб
3. **Контроль соблюдения правил безопасности** — контроль соблюдения правил безопасности движения

### Аналитические функции
1. **Анализ аварийности** — анализ причин аварий и нарушений
2. **Анализ рисков** — анализ рисков безопасности движения
3. **Разработка мер по повышению безопасности** — разработка мероприятий

### Контрольные функции
1. **Контроль безопасности движения** — мониторинг соблюдения правил безопасности
2. **Расследование нарушений** — расследование нарушений безопасности движения
3. **Контроль выполнения мер** — контроль выполнения мер по повышению безопасности

## Организационная структура

### Руководство
- **Директор Департамента** — возглавляет департамент, определяет политику безопасности
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный ревизор по безопасности движения** — контролирует соблюдение правил безопасности

### Отделы департамента
- **Отдел анализа аварийности** — анализирует причины аварий и нарушений
- **Отдел расследования нарушений** — проводит расследования нарушений
- **Отдел разработки мер** — разрабатывает меры по повышению безопасности
- **Отдел контроля** — контролирует выполнение мер безопасности

### Оперативный персонал
- **Старшие ревизоры** — проводят проверки и ревизии
- **Ревизоры по безопасности движения** — контролируют соблюдение правил
- **Инспекторы по безопасности** — проводят проверки и расследования

### Технический персонал
- **Инженеры по безопасности движения** — анализируют технические причины нарушений
- **Специалисты по анализу данных** — обрабатывают статистику

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или нарушении
- Оценка характера нарушения
- Определение необходимых мер

### 2. Принятие решений
- Принятие решений по реагированию на нарушение
- Определение состава комиссии по расследованию
- Назначение руководителя расследования

### 3. Проведение расследования
- Сбор информации о нарушении
- Анализ причин нарушения
- Разработка мер по предотвращению

### 4. Контроль выполнения мер
- Контроль выполнения мер по предотвращению
- Анализ эффективности мер
- Корректировка мер при необходимости

### 5. Завершение работ
- Проверка завершения работ
- Ведение документации
- Анализ причин нарушения
