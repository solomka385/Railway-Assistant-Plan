# НТЭ — Московская дирекция по энергообеспечению

## Общее описание

Московская дирекция по энергообеспечению (НТЭ) — структурное подразделение ОАО «РЖД», осуществляющее эксплуатацию, обслуживание и ремонт устройств электроснабжения железнодорожного транспорта. НТЭ обеспечивает бесперебойное электроснабжение контактной сети, тяговых подстанций и других устройств электроснабжения.

## Основные функции

### Эксплуатационные функции
1. **Эксплуатация контактной сети** — содержание и обслуживание контактной сети
2. **Эксплуатация тяговых подстанций** — содержание и обслуживание подстанций
3. **Эксплуатация линий электропередачи** — содержание и обслуживание ЛЭП
4. **Эксплуатация устройств освещения** — содержание и обслуживание освещения

### Ремонтные функции
1. **Текущий ремонт** — плановый ремонт устройств электроснабжения
2. **Капитальный ремонт** — полная замена оборудования
3. **Модернизация** — модернизация устройств электроснабжения

### Контрольные функции
1. **Контроль технического состояния** — мониторинг состояния устройств
2. **Контроль качества электроэнергии** — контроль параметров электроэнергии
3. **Анализ отказов** — анализ причин отказов устройств

## Организационная структура

### Руководство
- **Директор НТЭ** — возглавляет дирекцию, принимает ключевые решения
- **Первый заместитель директора** — замещает директора, курирует эксплуатацию
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническое состояние

### Дистанции электроснабжения
- **Начальники дистанций электроснабжения (ЭЧ)** — руководят дистанциями
- **Старшие электромеханики** — руководят бригадами
- **Электромеханики** — обслуживают устройства электроснабжения
- **Электромонтеры** — выполняют работы по обслуживанию

### Техническая служба
- **Начальник технического отдела** — руководит технической службой
- **Инженеры** — обеспечивают техническое сопровождение
- **Механики** — обслуживают технику

## Техническое оснащение

### Техника для обслуживания контактной сети
- **Вагоны-лаборатории контактной сети** — ВЛКС
- **Вышки контактной сети** — ВКС
- **Автодрезины** — для обслуживания контактной сети
- **Измерительные комплексы** — для диагностики контактной сети

### Техника для обслуживания подстанций
- **Передвижные подстанции** — для временного электроснабжения
- **Испытательные установки** — для испытания оборудования
- **Измерительные приборы** — для контроля параметров

### Транспорт
- **Автомобили** — для доставки персонала и материалов
- **Специальные кузова** — для перевозки оборудования

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или повреждении
- Оценка масштаба повреждений
- Определение необходимых сил и средств

### 2. Выдвижение к месту ЧС
- Формирование аварийных бригад
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Оценка обстановки
- Осмотр места повреждения
- Определение объема работ
- Разработка плана восстановления

### 4. Проведение восстановительных работ
- Устранение повреждений
- Восстановление электроснабжения
- Проверка качества работ

### 5. Завершение работ
- Открытие движения
- Составление отчета
- Анализ причин повреждения

## Взаимодействие с другими подразделениями

### ДИ — Московская дирекция инфраструктуры
- Координация работ по инфраструктуре
- Согласование планов работ
- Обмен информацией

### ДАВС — Дирекция аварийно-восстановительных средств
- Координация аварийно-восстановительных работ
- Обмен информацией о повреждениях
- Совместное планирование

### ДЦУП — Диспетчерский центр управления перевозками
- Согласование отключений контактной сети
- Координация движения при ремонтах
- Обмен информацией

### СЦБ — Устройства сигнализации, централизации и блокировки
- Координация работ по устройствам СЦБ
- Взаимодействие при ремонте
- Обеспечение электроснабжения устройств СЦБ

## Нормативные документы

1. **Правила технической эксплуатации железных дорог РФ** — основные требования к эксплуатации
2. **Инструкция по содержанию контактной сети** — определяет порядок содержания КС
3. **Правила устройства электроустановок** — определяют требования к электроустановкам
4. **Положение о системе технического обслуживания и ремонта** — определяет порядок ТОиР

## Показатели эффективности

1. **Состояние устройств электроснабжения** — оценка технического состояния
2. **Выполнение плана ремонтов** — процент выполненных ремонтных работ
3. **Количество отказов** — снижение количества отказов устройств
4. **Качество электроэнергии** — соответствие нормативным требованиям
