# МКБКС — Мобильный комплекс видео-конференц-связи

## Общее описание

Мобильный комплекс видео-конференц-связи (МКБКС) — специализированное подразделение для организации видеоконференцсвязи в полевых условиях при ликвидации аварий и чрезвычайных ситуаций. МКБКС обеспечивает оперативную видеосвязь между местом ЧС и координационными центрами.

## Основные функции

### Оперативные функции
1. **Организация видеоконференцсвязи** — обеспечение видеосвязи в зоне ЧС
2. **Трансляция видео** — трансляция видео с места происшествия
3. **Обеспечение каналов связи** — обеспечение стабильной работы каналов связи
4. **Техническая поддержка** — поддержка видеоконференций

### Плановые функции
1. **Подготовка к ЧС** — обучение персонала работе с оборудованием
2. **Техническое обслуживание** — обслуживание оборудования комплекса
3. **Планирование развертывания** — планирование развертывания комплекса
4. **Взаимодействие с другими службами** — координация работы

### Контрольные функции
1. **Контроль качества связи** — мониторинг качества видеосвязи
2. **Анализ работы оборудования** — анализ работы оборудования
3. **Разработка мер по повышению надежности** — разработка мероприятий

## Организационная структура

### Руководство
- **Начальник МКБКС** — возглавляет комплекс, отвечает за организацию работы
- **Заместитель начальника** — замещает начальника, курирует техническую часть

### Технический персонал
- **Инженер по видеосвязи** — настраивает и обслуживает оборудование
- **Техник по связи** — обеспечивает каналы связи
- **Оператор видеоконференцсвязи** — управляет оборудованием

### Водительский состав
- **Водитель спецавтомобиля** — обеспечивает доставку комплекса

## Техническое оснащение

### Спецавтомобиль
- **Спецавтомобиль с оборудованием** — мобильный комплекс видеосвязи
- **Автономный источник питания** — генератор для питания оборудования

### Оборудование видеосвязи
- **Спутниковые антенны** — для спутниковой связи
- **Видеокамеры** — для съемки
- **Микрофоны** — для аудиосвязи
- **Мониторы** — для отображения видео

### Оборудование связи
- **Спутниковый модем** — для спутниковой связи
- **Радиостанции** — для радиосвязи
- **Телефоны** — для телефонной связи

- Планирование маршрута

- Выдвижение к месту ЧС

- Настройка оборудования

### 3. Организация видеосвязи
- Установление связи с координационным центром
- Трансляция видео с места ЧС
- Техническая поддержка видеоконференций

### 4. Завершение работ
- Свертывание комплекса
- Ведение документации
- Анализ работы

## Показатели эффективности

1. **Время развертывания** — не более 30 минут
2. **Качество видеосвязи** — высокое качество видео и звука
3. **Надежность связи** — стабильная работа каналов связи
4. **Готовность к ЧС** — постоянная готовность к действиям
