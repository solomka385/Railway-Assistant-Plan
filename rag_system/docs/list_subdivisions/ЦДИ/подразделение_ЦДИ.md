# ЦДИ — Центральная дирекция инфраструктуры

## Общее описание

Центральная дирекция инфраструктуры (ЦДИ) — структурное подразделение ОАО «РЖД», осуществляющее управление эксплуатацией, обслуживанием и развитием инфраструктуры сети железных дорог. ЦДИ координирует работу дирекций инфраструктуры дорог, разрабатывает техническую политику и обеспечивает безопасность движения.

## Основные функции

### Руководящие функции
1. **Управление инфраструктурой** — общее руководство инфраструктурой сети дорог
2. **Координация работы дирекций инфраструктуры** — координация работы ДИ дорог
3. **Разработка технической политики** — разработка технической политики в области инфраструктуры
4. **Контроль технического состояния** — контроль технического состояния инфраструктуры

### Плановые функции
1. **Планирование развития инфраструктуры** — разработка планов развития
2. **Планирование ремонтов** — разработка планов ремонтных работ
3. **Планирование инвестиций** — планирование инвестиций в инфраструктуру

### Контрольные функции
1. **Контроль безопасности движения** — контроль соблюдения правил безопасности
2. **Контроль качества работ** — контроль качества ремонтных работ
3. **Анализ отказов** — анализ причин отказов устройств

## Организационная структура

### Руководство
- **Директор ЦДИ** — возглавляет дирекцию, принимает ключевые решения
- **Первый заместитель директора** — замещает директора, курирует эксплуатацию
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническую политику

### Функциональные управления
- **Управление пути** — курирует вопросы пути
- **Управление электроснабжения** — курирует вопросы электроснабжения
- **Управление СЦБ и связи** — курирует вопросы СЦБ и связи
- **Управление искусственных сооружений** — курирует вопросы искусственных сооружений

### Техническая служба
- **Начальник технического отдела** — руководит технической службой
- **Инженеры** — обеспечивают техническое сопровождение

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или повреждении
- Оценка масштаба повреждений
- Определение необходимых мер

### 2. Принятие решений
- Принятие решений по реагированию на ЧС
- Определение состава сил и средств
- Назначение руководителя ликвидации ЧС

### 3. Координация действий
- Координация действий дирекций инфраструктуры
- Координация взаимодействия с другими службами
- Контроль хода ликвидации ЧС

### 4. Информирование
- Информирование руководства ОАО «РЖД»
- Ведение документации

### 5. Завершение работ
- Проверка завершения работ
- Ведение документации
- Анализ причин повреждения
