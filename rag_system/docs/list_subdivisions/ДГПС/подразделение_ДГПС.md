# ДГПС — Старший дорожный диспетчер по управлению перевозками

## Общее описание

Старший дорожный диспетчер по управлению перевозками (ДГПС) — ключевое должностное лицо в системе управления перевозками, осуществляющее координацию и контроль движения поездов на уровне дороги. ДГПС обеспечивает безопасное и эффективное управление перевозочным процессом.

## Основные функции

### Оперативные функции
1. **Координация движения поездов** — управление движением на уровне дороги
2. **Контроль выполнения графика движения** — мониторинг соблюдения расписания
3. **Оперативное планирование** — корректировка планов перевозок при изменениях
4. **Координация работы диспетчерских служб** — взаимодействие с поездными и станционными диспетчерами

### Плановые функции
1. **Анализ выполнения плана перевозок** — оценка эффективности перевозочного процесса
2. **Разработка графиков движения** — участие в разработке расписаний
3. **Координация с другими дорогами** — взаимодействие с соседними дорогами
4. **Обучение диспетчерского персонала** — подготовка диспетчеров

## Организационная структура

### Руководство
- **Старший дорожный диспетчер** — возглавляет диспетчерскую службу дороги
- **Заместитель старшего дорожного диспетчера** — замещает диспетчера

### Диспетчерский персонал
- **Дорожные диспетчеры** — управляют движением на участках дороги
- **Диспетчеры по грузовым перевозкам** — координируют грузовые перевозки
- **Диспетчеры по пассажирским перевозкам** — координируют пассажирские перевозки
- **Диспетчеры по маневровой работе** — координируют маневровые операции

### Оперативный персонал
- **Оперативные дежурные** — ведут оперативную обстановку
- **Диспетчеры по вагонопотокам** — контролируют движение вагонов
- **Диспетчеры по локомотивам** — контролируют использование локомотивов

## Техническое оснащение

### Диспетчерские системы
- **Автоматизированная система управления движением (АСУ Д)** — основная система управления
- **Система мониторинга поездов** — отслеживание местоположения поездов
- **Система видеонаблюдения** — визуальный контроль станций и перегонов
- **Система связи** — радиосвязь, телефонная связь, спутниковая связь

### Информационные системы
- **Информационно-управляющая система** — обработка и передача информации
- **Система документооборота** — ведение оперативной документации
- **Система анализа данных** — анализ показателей перевозок

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или нарушении
- Оценка влияния на движение поездов
- Определение необходимых мер

### 2. Принятие оперативных решений
- Изменение графика движения поездов
- Направление поездов по обходным путям
- Остановка или задержка поездов при необходимости

### 3. Координация действий
- Взаимодействие с поездными диспетчерами
- Координация с аварийными службами
- Информирование руководства о ситуации

### 4. Восстановление нормального движения
- Постепенное восстановление графика
- Контроль хода восстановления
- Анализ причин нарушения

## Взаимодействие с другими подразделениями

### ЦД — Центральная дирекция управления движением
- Получение директив и указаний
- Отчет о выполнении плана перевозок
- Координация движения на сети дорог

### ДЦУП — Диспетчерский центр управления перевозками
- Взаимодействие по оперативным вопросам
- Координация движения поездов
- Обмен информацией

### ДЦС — Центр организации работы железнодорожных станций
- Координация работы станций
- Управление вагонопотоками
- Организация маневровой работы

### ДАВС — Дирекция аварийно-восстановительных средств
- Получение информации об авариях
- Координация движения при авариях
- Согласование времени открытия движения

### СЦБ — Устройства сигнализации, централизации и блокировки
- Координация при отказах устройств СЦБ
- Обеспечение безопасности движения
- Взаимодействие по техническим вопросам

## Нормативные документы

1. **Инструкция по диспетчерскому управлению движением поездов** — определяет порядок диспетчерского управления
2. **Правила технической эксплуатации** — определяют требования к безопасности движения
3. **График движения поездов** — основа для планирования перевозок
4. **Регламент взаимодействия** — определяет порядок взаимодействия с другими службами

## Показатели эффективности

1. **Выполнение графика движения** — процент поездов, прибывших по расписанию
2. **Объем перевозок** — количество перевезенных грузов и пассажиров
3. **Использование подвижного состава** — эффективность использования локомотивов и вагонов
4. **Безопасность движения** — отсутствие нарушений безопасности
