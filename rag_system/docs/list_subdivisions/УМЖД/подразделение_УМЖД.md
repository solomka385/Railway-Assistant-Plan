# УМЖД — Управление Московской железной дороги

## Общее описание

Управление Московской железной дороги (УМЖД) — территориальный орган управления ОАО «РЖД», осуществляющий руководство деятельностью Московской железной дороги. УМЖД координирует работу всех служб и подразделений дороги, обеспечивает выполнение плана перевозок и безопасность движения.

## Основные функции

### Руководящие функции
1. **Руководство деятельностью дороги** — общее руководство деятельностью дороги
2. **Координация работы служб** — координация работы всех служб дороги
3. **Принятие стратегических решений** — принятие решений по развитию дороги
4. **Контроль выполнения планов** — контроль выполнения планов перевозок

### Планирующие функции
1. **Планирование перевозок** — разработка планов грузовых и пассажирских перевозок
2. **Планирование развития** — разработка планов развития инфраструктуры
3. **Планирование инвестиций** — планирование инвестиций в развитие

### Контрольные функции
1. **Контроль безопасности движения** — контроль соблюдения правил безопасности
2. **Контроль качества услуг** — контроль качества предоставляемых услуг
3. **Анализ показателей работы** — анализ показателей работы дороги

## Организационная структура

### Руководство
- **Начальник Московской железной дороги** — возглавляет управление, принимает ключевые решения
- **Первый заместитель начальника** — замещает начальника, курирует ключевые направления
- **Заместители начальника** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническую политику

### Аппарат управления
- **Начальник штаба** — организует работу аппарата управления
- **Главный бухгалтер** — возглавляет бухгалтерскую службу
- **Начальник отдела кадров** — руководит кадровой работой
- **Начальник юридического отдела** — обеспечивает правовое сопровождение

### Функциональные отделы
- **Отдел эксплуатации** — координирует эксплуатационную деятельность
- **Отдел перевозок** — организует грузовые и пассажирские перевозки
- **Отдел инфраструктуры** — курирует содержание инфраструктуры
- **Отдел безопасности движения** — контролирует безопасность
- **Отдел экономики и финансов** — занимается экономическими вопросами

### Территориальные подразделения
- **Начальники отделений дороги** — руководят отделениями
- **Начальники станций** — руководят станциями
- **Начальники дистанций** — руководят техническими дистанциями
