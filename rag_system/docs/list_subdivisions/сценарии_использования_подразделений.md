# Сценарии использования подразделений железнодорожного транспорта

## Обзор документа

Данный документ содержит подробные сценарии использования подразделений железнодорожного транспорта при различных аварийных ситуациях. Каждый сценарий описывает последовательность действий, задействованные подразделения, используемую технику и ожидаемые результаты.

---

## Сценарий 1: Ликвидация последствий схода подвижного состава

### 1.1 Описание ситуации

**Тип аварии**: Сход вагонов с рельсов на перегоне
**Масштаб**: Средний (5-15 вагонов)
**Уровень угрозы**: Высокий
**Время восстановления**: 8-24 часа

### 1.2 Задействованные подразделения

| Подразделение | Роль | Время прибытия |
|---------------|------|----------------|
| ДАВС | Координация восстановительных работ | 15-30 минут |
| ДЦУП | Управление движением, остановка поездов | Мгновенно |
| ЦЧС | Мониторинг ситуации, прогнозирование | Мгновенно |
| ДИ | Восстановление пути и контактной сети | 30-60 минут |
| ДРП | Замена повреждённых рельсов и шпал | 30-60 минут |
| МЧ | Погрузка и вывоз обломков | 30-60 минут |
| МАБ | Медицинская помощь пострадавшим | 15-30 минут |
| НУЗ | Госпитализация тяжёлых больных | 30-60 минут |
| ФГП | Охрана места происшествия | 15-30 минут |
| ФГП "ВО ЖДТ России" | Охрана периметра | 15-30 минут |

### 1.3 Используемая техника

**ДАВС**:
- Кран КС-5371 — подъём вагонов
- Экскаватор Э-153 — расчистка завалов
- Бульдозер Т-170 — планировка территории
- Погрузчик Амкодор-332 — погрузка обломков
- Автомобиль КАМАЗ-5320 — транспортировка материалов
- Рация Motorola GP-340 — связь

**ДИ**:
- Автовышка — восстановление контактной сети
- Путевой инструмент — ремонт пути
- Диагностическое оборудование — проверка состояния

**ДРП**:
- Путеукладчик УК-25 — замена рельсов
- Щебнеочиститель — очистка балласта
- Выправочно-подбивочная машина — выправка пути

**МЧ**:
- Погрузчик фронтальный — погрузка обломков
- Экскаватор-погрузчик JCB 3CX — расчистка

**МАБ**:
- Реанимобиль класса С — первая помощь
- Дефибриллятор — реанимация
- Аппарат ИВЛ — искусственная вентиляция

### 1.4 Хронология действий

**Т0: Получение сигнала о сходе**
- ДЦУП получает сигнал от машиниста или системы мониторинга
- Автоматическая остановка всех поездов на участке
- Передача сигнала тревоги в ЦЧС и ДАВС

**Т0+5 мин: Первичная оценка**
- ЦЧС анализирует ситуацию по данным мониторинга
- ДЦУП информирует ДАВС о масштабах аварии
- Определение количества пострадавших

**Т0+15 мин: Выдвижение сил**
- ДАВС направляет аварийно-восстановительный поезд
- МАБ выезжает к месту аварии
- ФГП и ФГП "ВО ЖДТ России" выдвигают патрули

**Т0+30 мин: Прибытие первых бригад**
- Прибытие бригады ДАВС, начало оценки ситуации
- Прибытие МАБ, оказание первой помощи пострадавшим
- Оцепление места происшествия

**Т0+60 мин: Начало восстановительных работ**
- Прибытие бригад ДИ и ДРП
- Начало подъёма сошедших вагонов краном КС-5371
- Расчистка территории экскаватором Э-153

**Т0+120 мин: Активная фаза восстановления**
- Подъём всех вагонов
- Начало замены повреждённых рельсов и шпал
- Восстановление контактной сети

**Т0+240 мин (4 часа): Восстановление движения по одному пути**
- Завершение основных работ на одном пути
- Проверка состояния пути
- Открытие движения с ограниченной скоростью

**Т0+480 мин (8 часов): Полное восстановление**
- Завершение всех восстановительных работ
- Открытие движения в полном объёме
- Сдача объекта в эксплуатацию

**Т0+1440 мин (24 часа): Завершение работ**
- Полная уборка территории
- Составление акта о ликвидации аварии
- Анализ эффективности действий

### 1.5 Ключевые точки принятия решений

| Время | Решение | Ответственное подразделение |
|-------|---------|----------------------------|
| Т0 | Остановка движения на участке | ДЦУП |
| Т0+5 мин | Вызов аварийных бригад | ДАВС |
| Т0+15 мин | Определение масштабов работ | ДАВС, ЦЧС |
| Т0+60 мин | Начало подъёма вагонов | ДАВС |
| Т0+240 мин | Открытие движения по одному пути | ДЦУП, ДИ |
| Т0+480 мин | Полное открытие движения | ДЦУП |

### 1.6 Критерии успешной ликвидации

- Движение восстановлено в течение 8 часов
- Никто не пострадал или пострадавшие получили своевременную помощь
- Минимизирован экономический ущерб
- Территория полностью очищена
- Составлен акт о ликвидации аварии

## Сценарий 2: Ликвидация последствий обрушения моста

### 2.1 Описание ситуации

**Тип аварии**: Разрушение железнодорожного моста
**Масштаб**: Крупный (полное разрушение опоры)
**Уровень угрозы**: Критический
**Время восстановления**: 3-14 дней

### 2.2 Задействованные подразделения

| Подразделение | Роль | Время прибытия |
|---------------|------|----------------|
| ДАВС | Координация восстановительных работ | 30-60 минут |
| ДЦУП | Управление движением, перераспределение поездов | Мгновенно |
| ЦЧС | Мониторинг, прогнозирование | Мгновенно |
| ЦД | Стратегическое перераспределение поездопотоков | 15-30 минут |
| ДИ | Мостостроительные работы | 1-2 часа |
| ДРП | Восстановление подходов к мосту | 1-2 часа |
| МЧ | Расчистка завалов | 1-2 часа |
| БВС | Аэрофотосъёмка разрушений | 30-60 минут |
| МКБКС | Организация связи на месте | 40-60 минут |
| РСЧС | Координация на федеральном уровне | 30-60 минут |
| МЧС | Спасательные работы | 30-60 минут |
| УМЖД | Стратегическое руководство | 15-30 минут |

### 2.3 Используемая техника

**ДАВС**:
- Авиационная группа — доставка бригад
- Мобильный командный пункт — координация
- Краны КС-55713, РДК-25 — подъём конструкций
- Экскаваторы — земляные работы
- Спутниковые телефоны Thuraya — связь

**ДИ**:
- Мостостроительная техника — возведение опор
- Автокраны — монтаж конструкций
- Бетономешалки — приготовление раствора
- Сварочное оборудование — сборка конструкций

**БВС**:
- Дроны DJI Matrice 30T — аэрофотосъёмка
- Тепловизионные камеры — поиск пострадавших
- Лидарные системы — 3D-моделирование

**МКБКС**:
- Шасси КАМАЗ-6350 — мобильный командный пункт
- Системы спутниковой связи — видеоконференции
- Серверное оборудование — обработка данных

### 2.4 Хронология действий

**Т0: Получение сигнала об обрушении**
- ДЦУП получает сигнал от системы мониторинга или очевидцев
- Немедленная остановка всех поездов на подходах к мосту
- Передача сигнала в ЦЧС, ДАВС, УМЖД

**Т0+15 мин: Оценка ситуации**
- ЦЧС анализирует спутниковые снимки
- ДАВС оценивает масштаб разрушений
- УМЖД принимает решение о введении ЧС

**Т0+30 мин: Активация плана восстановления**
- ДАВС активирует план "Мост-202X"
- ЦД начинает перераспределение поездопотоков
- РСЧС информируется о ситуации

**Т0+60 мин: Выдвижение сил**
- Авиационная группа ДАВС вылетает к месту
- БВС запускает дроны для аэрофотосъёмки
- МКБКС выдвигается к месту аварии

**Т0+120 мин (2 часа): Прибытие первых сил**
- Прибытие авиационной группы
- Развертывание МКБКС
- Начало аэрофотосъёмки

**Т0+180 мин (3 часа): Планирование работ**
- Анализ данных аэрофотосъёмки
- Разработка плана восстановления
- Видеоконференция с руководством

**Т0+360 мин (6 часов): Начало работ**
- Прибытие мостостроительных бригад
- Начало расчистки подходов
- Подготовка площадки для временной опоры

**Т0+2880 мин (48 часов): Временное решение**
- Установка временных опор
- Прокладка временного полотна
- Открытие движения грузовых поездов

**Т0+10080 мин (7 дней): Основные работы**
- Возведение постоянных опор
- Монтаж пролётных строений
- Испытания конструкции

**Т0+20160 мин (14 дней): Полное восстановление**
- Завершение всех работ
- Открытие движения в полном объёме
- Сдача объекта в эксплуатацию

### 2.5 Ключевые точки принятия решений

| Время | Решение | Ответственное подразделение |
|-------|---------|----------------------------|
| Т0 | Остановка движения, введение ЧС | ДЦУП, УМЖД |
| Т0+30 мин | Активация плана восстановления | ДАВС |
| Т0+180 мин | Утверждение плана работ | УМЖД, ДАВС |
| Т0+2880 мин | Открытие временного движения | ДЦУП, ДИ |
| Т0+20160 мин | Полное открытие движения | ДЦУП, УМЖД |

### 2.6 Критерии успешной ликвидации

- Временное движение открыто в течение 48 часов
- Полное восстановление в течение 14 дней
- Минимизирован экономический ущерб
- Никто не пострадал
- Регион не был изолирован

## Сценарий 3: Ликвидация последствий пожара на вокзале

### 3.1 Описание ситуации

**Тип аварии**: Пожар в здании вокзала
**Масштаб**: Средний (угроза эвакуации 1000+ пассажиров)
**Уровень угрозы**: Высокий
**Время восстановления**: 2-6 часов

### 3.2 Задействованные подразделения

| Подразделение | Роль | Время прибытия |
|---------------|------|----------------|
| ДЦС | Координация эвакуации | Мгновенно |
| ДГПС | Принятие оперативных решений | Мгновенно |
| МЧС | Пожаротушение | 5-10 минут |
| МАБ | Медицинская помощь | 10-15 минут |
| НУЗ | Госпитализация пострадавших | 20-30 минут |
| ФГП | Охрана общественного порядка | 5-10 минут |
| ФГП "ВО ЖДТ России" | Помощь при эвакуации | 5-10 минут |
| ДЦУП | Управление движением поездов | Мгновенно |
| НТЭ | Восстановление электропитания | 15-30 минут |

### 3.3 Используемая техника

**МЧС**:
- Пожарные машины — тушение пожара
- Вертолёт Ми-8 — мониторинг с воздуха (при необходимости)
- Спасательное оборудование — эвакуация

**МАБ**:
- Реанимобиль — первая помощь
- Дефибриллятор — реанимация
- Носилки — эвакуация пострадавших

**ФГП**:
- Спецтранспорт — охрана периметра
- Средства связи — координация

**НТЭ**:
- Резервные генераторы — временное электропитание
- Диагностическое оборудование — поиск причин сбоя

### 3.4 Хронология действий

**Т0: Обнаружение пожара**
- Система пожарной сигнализации срабатывает
- ДЦС получает сигнал
- ДГПС принимает решение об эвакуации

**Т0+2 мин: Начало эвакуации**
- Включение системы оповещения
- Направление сотрудников к платформам
- Остановка всех поездов на станции

**Т0+5 мин: Вызов экстренных служб**
- Вызов МЧС, МАБ, ФГП
- Информирование ДЦУП о ситуации
- Перенаправление поездов на другие станции

**Т0+10 мин: Прибытие первых сил**
- Прибытие пожарных расчётов МЧС
- Начало тушения пожара
- Прибытие патрулей ФГП

**Т0+15 мин: Активная фаза эвакуации**
- Эвакуация всех пассажиров
- Прибытие МАБ
- Оказание помощи пострадавшим

**Т0+30 мин: Локализация пожара**
- Пожар локализован
- Начало эвакуации пострадавших в больницы
- Оценка ущерба

**Т0+60 мин: Ликвидация пожара**
- Пожар полностью потушен
- Начало проветривания
- Проверка на скрытые очаги

**Т0+120 мин (2 часа): Частичное восстановление**
- Восстановление электропитания
- Проверка систем безопасности
- Подготовка к возобновлению работы

**Т0+360 мин (6 часов): Полное восстановление**
- Вокзал готов к работе
- Движение восстановлено в полном объёме
- Составление акта о пожаре

### 3.5 Ключевые точки принятия решений

| Время | Решение | Ответственное подразделение |
|-------|---------|----------------------------|
| Т0 | Начало эвакуации | ДГПС, ДЦС |
| Т0+5 мин | Перенаправление поездов | ДЦУП |
| Т0+30 мин | Оценка масштабов ущерба | ДЦС, МЧС |
| Т0+120 мин | Частичное открытие вокзала | ДЦС |
| Т0+360 мин | Полное открытие вокзала | ДЦС, ДЦУП |

### 3.6 Критерии успешной ликвидации

- Никто не погиб
- Все пострадавшие получили своевременную помощь
- Эвакуация прошла организованно, без паники
- Пожар ликвидирован в течение 1 часа
- Вокзал восстановлен в течение 6 часов

## Сценарий 4: Ликвидация последствий обрыва контактной сети

### 4.1 Описание ситуации

**Тип аварии**: Обрыв контактной сети на электрифицированном участке
**Масштаб**: Малый (обрыв на 100-500 м)
**Уровень угрозы**: Средний
**Время восстановления**: 2-4 часа

### 4.2 Задействованные подразделения

| Подразделение | Роль | Время прибытия |
|---------------|------|----------------|
| ДЦУП | Управление движением | Мгновенно |
| ЦУЭД | Координация энергообеспечения | Мгновенно |
| ДИ | Восстановление контактной сети | 20-40 минут |
| НТЭ | Проверка подстанций | 20-40 минут |
| ЦТО | Диагностика причин обрыва | 30-60 минут |
| ДЦС | Информирование пассажиров | Мгновенно |

### 4.3 Используемая техника

**ДИ**:
- Автовышка — восстановление контактной сети
- Специальный инструмент — монтаж проводов
- Диагностическое оборудование — проверка состояния

**НТЭ**:
- Диагностическое оборудование — проверка подстанций
- Резервные генераторы — временное питание

**ЦТО**:
- Диагностические стенды — анализ причин
- Специализированное ПО — моделирование ситуации

### 4.4 Хронология действий

**Т0: Обнаружение обрыва**
- Система мониторинга фиксирует обрыв
- ДЦУП получает сигнал
- Автоматическое отключение питания участка

**Т0+2 мин: Остановка движения**
- ДЦУП останавливает все электропоезда на участке
- Информирование ДЦС для оповещения пассажиров
- Передача сигнала в ЦУЭД и ДИ

**Т0+10 мин: Оценка ситуации**
- ЦУЭД оценивает состояние сети
- ДИ направляет бригаду на место
- НТЭ проверяет подстанции

**Т0+30 мин: Прибытие бригады**
- Прибытие бригады ДИ
- Оценка повреждений
- Подготовка к восстановительным работам

**Т0+60 мин: Начало восстановления**
- Начало монтажа контактной сети
- Проверка состояния опор
- Подготовка проводов

**Т0+120 мин (2 часа): Основные работы**
- Завершение монтажа проводов
- Проверка натяжения
- Подключение к сети

**Т0+180 мин (3 часа): Испытания**
- Подача напряжения
- Проверка работоспособности
- Пробный пропуск поезда

**Т0+240 мин (4 часа): Полное восстановление**
- Открытие движения в полном объёме
- Составление акта о восстановлении

### 4.5 Ключевые точки принятия решений

| Время | Решение | Ответственное подразделение |
|-------|---------|----------------------------|
| Т0 | Остановка движения, отключение питания | ДЦУП, ЦУЭД |
| Т0+10 мин | Направление бригады ДИ | ДИ, ЦУЭД |
| Т0+180 мин | Подача напряжения | НТЭ, ЦУЭД |
| Т0+240 мин | Открытие движения | ДЦУП |

### 4.6 Критерии успешной ликвидации

- Движение восстановлено в течение 4 часов
- Причина обрыва выявлена и устранена
- Никто не пострадал
- Минимизированы задержки поездов

## Сценарий 5: Ликвидация последствий разлива опасных химических веществ

### 5.1 Описание ситуации

**Тип аварии**: Разлив опасных химических веществ из цистерны
**Масштаб**: Средний (угроза загрязнения окружающей среды)
**Уровень угрозы**: Высокий
**Время восстановления**: 6-24 часа

### 5.2 Задействованные подразделения

| Подразделение | Роль | Время прибытия |
|---------------|------|----------------|
| ДАВС | Координация работ | 15-30 минут |
| ЦЧС | Моделирование распространения | Мгновенно |
| МЧС | Ликвидация разлива | 15-30 минут |
| НЦОП | Экологический мониторинг | 30-60 минут |
| НУЗ | Медицинская помощь | 20-40 минут |
| РСЧС | Координация на федеральном уровне | 30-60 минут |
| ДЦУП | Управление движением | Мгновенно |
