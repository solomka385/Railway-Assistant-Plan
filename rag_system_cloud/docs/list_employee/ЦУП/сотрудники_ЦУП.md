# Сотрудники ЦУП (Центр управления перевозками)

## Общая информация

Центр управления перевозками (ЦУП) — координационный центр, осуществляющий оперативное управление перевозочным процессом на сети железных дорог.

## Организационная структура

### Руководство
- **Начальник ЦУП** — возглавляет центр, принимает ключевые решения по управлению перевозками
- **Первый заместитель начальника** — замещает начальника, курирует оперативную работу
- **Заместители начальника** — курируют отдельные направления деятельности
- **Главный диспетчер** — руководит диспетчерской службой

### Диспетчерская служба
- **Главный диспетчер по перевозкам** — координирует перевозки на сети дорог
- **Диспетчер по грузовым перевозкам** — управляет грузовыми перевозками
- **Диспетчер по пассажирским перевозкам** — управляет пассажирскими перевозками
- **Диспетчер по маневровой работе** — координирует маневровые операции

### Оперативный персонал
- **Оперативный дежурный** — ведет оперативную обстановку
- **Диспетчер по вагонопотокам** — контролирует движение вагонов
- **Диспетчер по локомотивам** — контролирует использование локомотивов

### Аналитический отдел
- **Начальник аналитического отдела** — руководит аналитической работой
- **Аналитик по перевозкам** — анализирует показатели перевозок
- **Экономист** — рассчитывает экономические показатели

## Требования к сотрудникам

### Квалификационные требования
- Высшее образование (техническое, экономическое)
- Опыт работы на железнодорожном транспорте не менее 2 лет
- Наличие удостоверения диспетчера
- Знание организации перевозочного процесса

### Профессиональные навыки
- Навыки оперативного управления перевозками
- Умение работать в стрессовых ситуациях
- Знание технологии перевозочного процесса
- Навыки работы с автоматизированными системами

## График работы

- **Режим дежурств**: круглосуточное дежурство
- **Продолжительность смены**: 12 часов
- **График сменности**: 2/2 (два дня работы, два дня отдыха)

## Взаимодействие с другими подразделениями

### Основные контакты
- **ЦД** — Центральная дирекция управления движением
- **ДЦУП** — диспетчерский центр управления перевозками
- **ДЦС** — центр организации работы станций
- **ДАВС** — координация при авариях
- **Все дирекции** — координация перевозок

## Обязанности при ЧС

1. Координация перевозок в аварийной ситуации
2. Перераспределение вагонопотоков
3. Организация обходных маршрутов
4. Координация работы станций и диспетчерских служб
5. Взаимодействие с аварийными службами
6. Ведение оперативной документации

## Полномочия

- Изменение планов перевозок
- Перераспределение подвижного состава
- Координация работы всех служб перевозок
- Принятие оперативных решений
