# Сотрудники ЦУТСС (Центр управления технологической сетью связи)

## Общая информация

Центр управления технологической сетью связи (ЦУТСС) — технический центр, обеспечивающий управление и мониторинг технологической сети связи железнодорожного транспорта.

## Организационная структура

### Руководство
- **Начальник ЦУТСС** — возглавляет центр, отвечает за работоспособность сети связи
- **Заместитель начальника** — замещает начальника, курирует техническую часть
- **Главный инженер** — отвечает за техническую политику

### Оперативный персонал
- **Дежурный инженер** — контролирует работоспособность сети связи
- **Старший смены** — руководит дежурной сменой
- **Оператор сети связи** — управляет каналами связи
- **Диспетчер связи** — координирует работу бригад

### Технический персонал
- **Инженер по мониторингу** — осуществляет мониторинг сети
- **Инженер по эксплуатации** — обслуживает оборудование сети
- **Инженер по ремонту** — ремонтирует оборудование
- **Техник связи** — выполняет работы по обслуживанию

### Административный персонал
- **Администратор баз данных** — управляет базами данных
- **Специалист по информационной безопасности** — обеспечивает безопасность сети

## Требования к сотрудникам

### Квалификационные требования
- Высшее или среднее техническое образование
- Опыт работы в сфере связи не менее 1 года
- Наличие удостоверения по специальности
- Знание оборудования сетей связи

### Профессиональные навыки
- Владение системами мониторинга сетей связи
- Навыки диагностики и устранения неисправностей
- Знание протоколов передачи данных
- Умение работать с автоматизированными системами

## График работы

- **Режим дежурств**: круглосуточное дежурство
- **Продолжительность смены**: 12 часов
- **График сменности**: 2/2 (два дня работы, два дня отдыха)

## Взаимодействие с другими подразделениями

### Основные контакты
- **ЦСС** — Центральная станция связи
- **ЦТУ** — отдел технического управления сетями связи
- **МКБКС** — мобильный комплекс видеосвязи
- **ДАВС** — координация при авариях
- **НТЭ** — энергообеспечение оборудования связи

## Обязанности при ЧС

1. Мониторинг состояния сети связи в зоне ЧС
2. Координация восстановления каналов связи
3. Обеспечение резервных каналов связи
4. Взаимодействие с аварийными службами
5. Ведение журналов отказов и ремонтов
6. Анализ причин отказов

## Специальное оборудование

- Системы мониторинга сетей связи
- Автоматизированные рабочие места
- Средства радиосвязи
- Запасное оборудование
- Системы резервного питания
