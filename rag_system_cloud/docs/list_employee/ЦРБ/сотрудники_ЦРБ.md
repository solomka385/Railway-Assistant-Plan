# Сотрудники ЦРБ (Департамент безопасности движения ОАО «РЖД»)

## Общая информация

Департамент безопасности движения ОАО «РЖД» (ЦРБ) — структурное подразделение, обеспечивающее безопасность движения поездов на сети железных дорог, анализ причин нарушений и разработку мер по их предотвращению.

## Организационная структура

### Руководство
- **Директор Департамента** — возглавляет департамент, определяет политику безопасности движения
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный ревизор по безопасности движения** — контролирует соблюдение правил безопасности

### Отделы департамента
- **Отдел анализа аварийности** — анализирует причины аварий и нарушений
- **Отдел расследования нарушений** — проводит расследования нарушений безопасности движения
- **Отдел разработки мер** — разрабатывает меры по повышению безопасности
- **Отдел контроля** — контролирует выполнение мер безопасности

### Оперативный персонал
- **Старший ревизор** — проводит проверки и ревизии
- **Ревизор по безопасности движения** — контролирует соблюдение правил
- **Инспектор по безопасности** — проводит проверки и расследования

### Технический персонал
- **Инженер по безопасности движения** — анализирует технические причины нарушений
- **Специалист по анализу данных** — обрабатывает статистику

## Требования к сотрудникам

### Квалификационные требования
- Высшее техническое образование
- Опыт работы на железнодорожном транспорте не менее 3 лет
- Знание правил технической эксплуатации
- Опыт работы в сфере безопасности движения

### Профессиональные навыки
- Знание правил безопасности движения
- Навыки расследования нарушений
- Умение анализировать причины аварий
- Навыки разработки мер по повышению безопасности

## График работы

- **Режим работы**: пятидневная рабочая неделя
- **Часы работы**: 9:00 - 18:00
- **Дежурства**: круглосуточное дежурство оперативного персонала

## Взаимодействие с другими подразделениями

### Основные контакты
- **ЦД** — Центральная дирекция управления движением
- **ЦДИ** — Центральная дирекция инфраструктуры
- **ЦБЗ** — Департамент безопасности
- **ДАВС** — координация при авариях
- **Все дирекции** — контроль безопасности движения

## Обязанности при ЧС

1. Анализ причин возникновения ЧС
2. Расследование нарушений безопасности движения
3. Разработка мер по предотвращению подобных ситуаций
4. Контроль выполнения мер безопасности
5. Взаимодействие с другими службами
6. Ведение документации по нарушениям

## Полномочия

- Проведение проверок и ревизий
- Приостановка работы при нарушениях безопасности
- Требование устранения нарушений
- Разработка обязательных мер безопасности
