# Сотрудники ЦДИ — Центральная дирекция инфраструктуры

## Обзор

Данный документ содержит информацию о должностях и сотрудниках Центральной дирекции инфраструктуры — подразделения, отвечающего за содержание и обслуживание железнодорожной инфраструктуры на уровне сети железных дорог.

---

## Должности ЦДИ

### 14.1 Инженер по эксплуатации инфраструктуры
- **Подразделение**: ЦДИ
- **Квалификация**: Высшее техническое образование, опыт работы не менее 3 лет
- **Обязанности**:
  - Контроль состояния инфраструктуры
  - Планирование ремонтных работ
  - Координация работы дирекций инфраструктуры
- **Используемая техника**:
  - Рация Motorola GP-340
  - Ноутбук
  - Измерительные приборы
- **Примеры использования**:
  - Контроль состояния инфраструктуры
  - Планирование ремонтных работ
  - Координация работы дирекций

### 14.2 Инженер по диагностике
- **Подразделение**: ЦДИ
- **Квалификация**: Высшее техническое образование, опыт работы не менее 2 лет
- **Обязанности**:
  - Диагностика инфраструктуры
  - Анализ данных мониторинга
  - Выявление неисправностей
- **Используемая техника**:
  - Рация Motorola GP-340
  - Ноутбук с диагностическим ПО
  - Измерительные приборы
- **Примеры использования**:
  - Диагностика состояния пути
  - Анализ данных мониторинга
  - Выявление потенциальных неисправностей

---

## Организационная структура

```
Начальник ЦДИ
├── Заместители
├── Служба эксплуатации
│   └── Инженеры по эксплуатации
└── Служба диагностики
    └── Инженеры по диагностике
```

---

## Требования к персоналу

### Общие требования
- Готовность к работе в любое время суток
- Знание правил техники безопасности
- Умение работать с диагностическим оборудованием

### Специальные требования
- Наличие допусков к работе с электрооборудованием
- Прохождение медицинского осмотра

---

## Режим работы

- **Дежурный персонал**: Круглосуточное дежурство при ЧС
- **Технический персонал**: График 5/2 с возможностью вызова

---

## Взаимодействие с другими подразделениями

ЦДИ взаимодействует со следующими подразделениями:
- **ДАВС** — координация аварийно-восстановительных работ
- **ДИ** — координация работ на путях
- **НТЭ** — координация энергообеспечения
- **ЦЧС** — обмен информацией о ситуациях
