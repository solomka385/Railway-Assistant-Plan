# МЧС — Министерство Российской Федерации по делам гражданской обороны, чрезвычайным ситуациям и ликвидации последствий стихийных бедствий

## Общее описание

Министерство Российской Федерации по делам гражданской обороны, чрезвычайным ситуациям и ликвидации последствий стихийных бедствий (МЧС России) — федеральный орган исполнительной власти, осуществляющий функции по выработке и реализации государственной политики, нормативно-правовому регулированию, а также по надзору и контролю в области гражданской обороны, защиты населения и территорий от чрезвычайных ситуаций природного и техногенного характера, обеспечения пожарной безопасности и безопасности людей на водных объектах.

## Основные функции

### Оперативные функции
1. **Ликвидация чрезвычайных ситуаций** — проведение аварийно-спасательных работ
2. **Поиск и спасение пострадавших** — поиск и спасение людей при ЧС
3. **Тушение пожаров** — тушение пожаров на объектах и транспорте
4. **Эвакуация населения** — эвакуация населения из зон ЧС

### Плановые функции
1. **Прогнозирование ЧС** — прогнозирование возникновения ЧС
2. **Предупреждение ЧС** — разработка мер по предупреждению ЧС
3. **Подготовка населения** — обучение населения действиям при ЧС
4. **Взаимодействие с другими службами** — координация работы

### Контрольные функции
1. **Контроль готовности** — контроль готовности к ЧС
2. **Надзор за соблюдением правил** — надзор за соблюдением правил пожарной безопасности
3. **Анализ ЧС** — анализ причин ЧС и разработка мер
