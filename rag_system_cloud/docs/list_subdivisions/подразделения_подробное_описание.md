# Подробный справочник подразделений железнодорожного транспорта

## Обзор документа

Данный документ содержит подробное описание всех подразделений железнодорожного транспорта, их функций, обязанностей, используемой техники и примеров взаимодействия при ликвидации аварийных ситуаций.

---

## 1. ДАВС — Дирекция аварийно-восстановительных средств

### 1.1 Общее описание

ДАВС — это центральное стратегическое подразделение ОАО "РЖД", отвечающее за координацию, обеспечение ресурсами и управление всеми силами, задействованными в ликвидации последствий аварий и чрезвычайных ситуаций на всей сети железных дорог России.

### 1.2 Структура и ресурсы

- **47 аварийно-восстановительных поездов (АВП)**, размещённых в ключевых узлах страны
- **12 мобильных командных пунктов** на базе КАМАЗ
- **3 резервных склада** запасных частей и материалов
- **5 авиационных групп** для доставки бригад в труднодоступные регионы
- **200 единиц специализированной техники** — краны, экскаваторы, пожарные машины, генераторы

### 1.3 Основные функции

- Координация всех аварийно-восстановительных работ
- Управление ресурсами и техникой
- Взаимодействие с МЧС России, Минобороны, ФСБ, правительствами регионов
- Планирование и проведение учений
- Анализ эффективности восстановительных работ

### 1.4 Используемая техника

- Краны КС-5371, КС-55713, РДК-25
- Экскаваторы Э-153, JCB 3CX
- Бульдозеры Т-170
- Погрузчики Амкодор-332
- Автомобили КАМАЗ-5320
- Дрезины АГВ, мотовозы МПТ
- Рации Motorola GP-340, спутниковые телефоны Thuraya

### 1.5 Взаимодействие с другими подразделениями

ДАВС взаимодействует со следующими подразделениями:
- **МЧС России** — пожаротушение, спасение людей
- **РСЧС** — координация при ЧС
- **ЦЧС** — ситуационный мониторинг
- **ЦД** — управление движением
- **ЦСС** — обеспечение связи
- **ФГП "ВО ЖДТ России"** — охрана объектов
- **НУЗ** — медицинская помощь
- **ЦРБ** — медицинские учреждения
- **ДЦУП** — диспетчерское управление
- **ДЦС** — управление станциями
- **ФГП** — федеральная государственная полиция

### 1.6 Примеры использования

#### Пример 1: Разрушение железнодорожного моста после наводнения
**Дата**: 3 августа 2023 года
**Место**: Краснодарский край, мост через реку Кубань
**Ситуация**: В результате сильнейшего наводнения мост был размыт, движение полностью остановлено, угроза изоляции региона.

**Задействованные подразделения**:
- ДАВС — координация, АВП №7, авиагруппа
- МЧС — спасательные работы
- РСЧС — координация на региональном уровне
- ЦЧС — мониторинг ситуации
- ЦД — перераспределение поездопотоков
- ДИ — мостостроительные бригады

**Хронология**:
1. 00:00 — Получение сигнала о разрушении моста
2. 00:15 — Активация плана "Мост-2023"
3. 01:00 — Выдвижение АВП №7 и авиагруппы
4. 06:00 — Прибытие первых бригад, начало расчистки подходов
5. 48:00 — Установка временных опор
6. 72:00 — Прокладка временного полотна, движение грузовых поездов возобновлено
7. 14 дней — Полное восстановление моста

#### Пример 2: Сход вагонов с рельсов на перегоне
**Дата**: 12 марта 2024 года
**Место**: Перегон "Бронницы — Раменское"
**Ситуация**: Сошли 15 вагонов грузового состава с углём, перекрыли оба пути, повреждена контактная сеть.

**Задействованные подразделения**:
- ДАВС — координация, три бригады АВП
- МЧ — погрузка и вывоз угля
- ДРП — замена повреждённых рельсов и шпал
- ЦТО — восстановление контактной сети
- ДЦУП — управление движением
- ЦЧС — мониторинг

**Хронология**:
1. 00:00 — Получение сигнала о сходе
2. 00:15 — Направление трёх бригад АВП
3. 00:30 — Прибытие первой бригады
4. 08:00 — Движение по одному пути восстановлено
5. 14:00 — Движение по второму пути восстановлено

## 2. ЦЧС — Ситуационный центр мониторинга ЧС

### 2.1 Общее описание

ЦЧС — единый центр сбора и анализа данных о чрезвычайных ситуациях. Использует искусственный интеллект, Big Data, спутниковые снимки и датчики для прогнозирования развития ситуаций.

### 2.2 Основные функции

- Сбор и анализ данных о ЧС в реальном времени
- Прогнозирование развития аварийных ситуаций
- Оповещение соответствующих служб
- Координация действий при ликвидации ЧС
- Мониторинг состояния инфраструктуры

### 2.3 Используемая техника

- Спутниковые системы мониторинга
- Беспилотные летательные аппараты
- Системы видеонаблюдения
- Программное обеспечение для анализа Big Data
- Системы ИИ-аналитики

### 2.4 Взаимодействие с другими подразделениями

ЦЧС взаимодействует со следующими подразделениями:
- **ДАВС** — передача данных о ЧС
- **МЧС** — координация спасательных работ
- **РСЧС** — федеральная координация
- **ЦД** — информирование о ситуации на путях
- **ДЦУП** — передача данных для управления движением
- **БВС** — использование дронов для мониторинга

### 2.5 Примеры использования

#### Пример 1: Сход лавины на пути в горной местности
**Дата**: 15 января 2024 года
**Место**: Горный участок Кавказского направления
**Ситуация**: Сошла лавина, перекрывшая железнодорожный путь.

**Задействованные подразделения**:
- ЦЧС — мониторинг, прогнозирование
- ДАВС — координация восстановительных работ
- РСЧС — координация на федеральном уровне

**Действия ЦЧС**:
1. Получение данных от датчиков о сходе лавины
2. Анализ спутниковых снимков
3. Оценка объёма снежной массы
4. Прогнозирование возможных повторных сходов
5. Передача данных в ДАВС и ДЦУП

#### Пример 2: Разлив опасных химических веществ
**Дата**: 20 сентября 2024 года
**Место**: Перегон в Тульской области
**Ситуация**: Разлив химических веществ из цистерны.

**Задействованные подразделения**:

- ДАВС — координация
- РСЧС — координация
- НУЗ — медицинская помощь

**Действия ЦЧС**:
