# ДИ ЦУСИ — Центр управления содержанием инфраструктуры

## Общее описание

Центр управления содержанием инфраструктуры (ДИ ЦУСИ) — структурное подразделение Московской дирекции инфраструктуры, осуществляющее централизованное управление и координацию работ по содержанию и ремонту инфраструктуры. ЦУСИ обеспечивает эффективное планирование и контроль выполнения работ по обслуживанию железнодорожной инфраструктуры.

## Основные функции

### Планирующие функции
1. **Планирование работ по содержанию инфраструктуры** — разработка планов ТОиР
2. **Координация ремонтных работ** — согласование сроков и объемов работ
3. **Управление ресурсами** — распределение техники и персонала
4. **Контроль выполнения планов** — мониторинг хода работ

### Контрольные функции
1. **Контроль технического состояния** — мониторинг состояния инфраструктуры
2. **Анализ отказов** — анализ причин отказов устройств
3. **Контроль качества работ** — оценка качества выполненных работ
4. **Аудит деятельности дистанций** — проверка работы дистанций

### Аналитические функции
1. **Анализ показателей работы** — оценка эффективности работы
2. **Прогнозирование отказов** — прогнозирование возможных отказов
3. **Разработка мер по повышению надежности** — разработка мероприятий
4. **Оптимизация процессов** — оптимизация процессов содержания

## Организационная структура

### Руководство
- **Начальник ЦУСИ** — возглавляет центр, принимает ключевые решения
- **Заместитель начальника** — замещает начальника, курирует планирование
- **Главный инженер** — отвечает за техническую часть

### Отдел планирования
- **Начальник отдела планирования** — руководит планированием работ
- **Плановики** — составляют планы работ
- **Экономисты** — рассчитывают стоимость работ

### Отдел контроля
- **Начальник отдела контроля** — руководит контролем
- **Инспекторы** — проводят проверки
- **Аудиторы** — проводят аудит

### Отдел анализа
- **Начальник отдела анализа** — руководит аналитической работой
- **Аналитики** — анализируют показатели
- **Специалисты по надежности** — анализируют надежность

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или повреждении
- Оценка влияния на инфраструктуру
- Определение необходимых мер

### 2. Координация восстановительных работ
- Определение состава бригад
- Координация действий дистанций
- Контроль хода работ

### 3. Анализ причин
- Анализ причин повреждения
- Разработка мер по предотвращению
- Внесение изменений в планы

### 4. Контроль восстановления
- Проверка качества восстановительных работ
- Контроль открытия движения
- Составление отчета
