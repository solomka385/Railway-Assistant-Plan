# ДРП — Московская дирекция по ремонту пути

## Общее описание

Московская дирекция по ремонту пути (ДРП) — структурное подразделение ОАО «РЖД», осуществляющее капитальный и средний ремонт железнодорожного пути на Московской железной дороге. ДРП обеспечивает качественное выполнение ремонтных работ для поддержания инфраструктуры в исправном состоянии.

## Основные функции

### Ремонтные функции
1. **Капитальный ремонт пути** — полная замена верхнего строения пути
2. **Средний ремонт пути** — частичная замена элементов пути
3. **Ремонт искусственных сооружений** — ремонт мостов, тоннелей, путепроводов
4. **Ремонт земляного полотна** — укрепление и восстановление земляного полотна

### Подготовительные функции
1. **Подготовка к ремонту** — подготовка материалов и техники
2. **Планирование ремонтов** — разработка планов ремонтных работ
3. **Координация с другими службами** — согласование сроков и объемов работ
4. **Обеспечение безопасности** — обеспечение безопасности при ремонтах

### Контрольные функции
1. **Контроль качества работ** — оценка качества выполненных работ
2. **Приемка работ** — приемка выполненных работ
3. **Анализ дефектов** — анализ дефектов и разработка мер по их устранению

## Организационная структура

### Руководство
- **Директор ДРП** — возглавляет дирекцию, принимает ключевые решения
- **Первый заместитель директора** — замещает директора, курирует производство
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническую часть

### Путевые машинные станции (ПМС)
- **Начальники ПМС** — руководят путевыми машинными станциями
- **Дорожные мастера** — руководят бригадами
- **Монтеры пути** — выполняют путевые работы

### Мостопоезда
- **Начальники мостопоездов** — руководят мостопоездами
- **Мастера** — руководят бригадами
- **Рабочие** — выполняют работы по ремонту сооружений

### Техническая служба
- **Начальник технического отдела** — руководит технической службой
- **Инженеры** — обеспечивают техническое сопровождение
- **Механики** — обслуживают технику

## Техническое оснащение

### Путевая техника
- **Путеукладочные краны** — УК-25, УК-25/9-18, УК-25/9-21
- **Выправочно-подбивочно-отделочные машины** — ВПО-3000, ВПР-02
- **Балластоочистительные машины** — БМ, СЧ-600, ЩОМ-4, ЩОМ-6
- **Рельсосварочные поезда** — ПРСМ, ПРСМ-2
- **Дрезины и автомотрисы** — для доставки персонала и материалов

### Техника для ремонта сооружений
- **Краны** — для подъемных работ
- **Экскаваторы** — для земляных работ
- **Бульдозеры** — для планировочных работ
- **Бетонные заводы** — для приготовления бетона

### Транспорт
- **Платформы** — для перевозки материалов
- **Полувагоны** — для перевозки балласта
- **Крытые вагоны** — для перевозки материалов

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или повреждении пути
- Оценка масштаба повреждений
- Определение необходимости участия в восстановительных работах

### 2. Выдвижение к месту ЧС
- Формирование ремонтных бригад
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Оценка обстановки
- Осмотр места повреждения
- Определение объема работ
- Разработка плана восстановления

### 4. Проведение восстановительных работ
- Устранение повреждений
- Восстановление пути
- Проверка качества работ

### 5. Завершение работ
- Открытие движения
- Составление отчета
- Анализ причин повреждения

## Взаимодействие с другими подразделениями

### ДИ — Московская дирекция инфраструктуры
- Координация ремонтных работ
- Согласование планов ремонтов
- Обмен информацией о состоянии пути

### ДАВС — Дирекция аварийно-восстановительных средств
- Участие в аварийно-восстановительных работах
- Координация действий при авариях
- Обмен информацией

### ДЦУП — Диспетчерский центр управления перевозками
- Согласование «окон» в движении
- Координация движения при ремонтах
- Обмен информацией

### СЦБ — Устройства сигнализации, централизации и блокировки
- Координация работ по устройствам СЦБ
- Взаимодействие при ремонте
- Обеспечение безопасности движения

## Нормативные документы

1. **Правила технической эксплуатации железных дорог РФ** — основные требования к эксплуатации
2. **Инструкция по капитальному ремонту пути** — определяет порядок капитального ремонта
3. **Технические условия на ремонт пути** — определяют требования к качеству
4. **План ремонтных работ** — план ремонтных работ на год

## Показатели эффективности

1. **Выполнение плана ремонтов** — процент выполненных ремонтных работ
2. **Качество работ** — оценка качества выполненных работ
3. **Соблюдение сроков** — выполнение работ в установленные сроки
4. **Безопасность работ** — отсутствие нарушений безопасности
