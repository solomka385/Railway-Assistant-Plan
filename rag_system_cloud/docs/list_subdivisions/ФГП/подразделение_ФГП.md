# ФГП — Федеральное государственное предприятие «Ведомственная охрана железнодорожного транспорта Российской Федерации»

## Общее описание

Федеральное государственное предприятие «Ведомственная охрана железнодорожного транспорта Российской Федерации» (ФГП) — специализированное подразделение, осуществляющее охрану объектов, грузов и имущества железнодорожного транспорта. ФГП обеспечивает безопасность на железнодорожном транспорте и взаимодействует с правоохранительными органами.

## Основные функции

### Охранительные функции
1. **Охрана объектов** — охрана железнодорожных объектов
2. **Охрана грузов** — охрана грузов в пути следования
3. **Охрана подвижного состава** — охрана подвижного состава
4. **Сопровождение грузов** — сопровождение ценных грузов

### Контрольные функции
1. **Контроль доступа** — контроль доступа на объекты
2. **Контроль соблюдения правил** — контроль соблюдения правил безопасности
3. **Пресечение нарушений** — пресечение нарушений безопасности

### Взаимодействие с правоохранительными органами
1. **Взаимодействие с МВД** — взаимодействие с органами внутренних дел
2. **Взаимодействие с ФСБ** — взаимодействие с Федеральной службой безопасности
3. **Взаимодействие с прокуратурой** — взаимодействие с прокуратурой

## Организационная структура

### Руководство
- **Начальник ФГП** — возглавляет предприятие, принимает ключевые решения
- **Заместители начальника** — курируют отдельные направления деятельности
- **Начальник штаба** — организует планирование и координацию

### Оперативные подразделения
- **Начальники отрядов** — руководят отрядами ведомственной охраны
- **Старшие смен** — руководят работой смен
- **Сотрудники охраны** — осуществляют охрану объектов

### Технический персонал
- **Инженеры по техническим средствам охраны** — обслуживают системы охраны
- **Техники** — обслуживают оборудование

## Порядок действий при ЧС

### 1. Получение информации о нарушении
- Принятие сигнала о нарушении
- Оценка характера нарушения
- Определение необходимых мер

### 2. Выдвижение к месту нарушения
- Формирование группы реагирования
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Пресечение нарушения
- Пресечение нарушения
- Задержание нарушителей (при необходимости)
- Охрана места происшествия

### 4. Взаимодействие с правоохранительными органами
- Передача информации правоохранительным органам
- Содействие в расследовании
- Ведение документации

### 5. Завершение работ
- Проверка завершения работ
- Ведение документации
- Анализ причин нарушения
