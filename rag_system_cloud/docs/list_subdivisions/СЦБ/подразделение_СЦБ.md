# СЦБ — Устройства сигнализации, централизации и блокировки

## Общее описание

Устройства сигнализации, централизации и блокировки (СЦБ) — техническая служба, обеспечивающая эксплуатацию, обслуживание и ремонт систем железнодорожной автоматики и телемеханики. СЦБ обеспечивает безопасность движения поездов и эффективность управления движением.

## Основные функции

### Эксплуатационные функции
1. **Эксплуатация устройств СЦБ** — содержание и обслуживание устройств СЦБ
2. **Эксплуатация систем АЛСН** — содержание и обслуживание автоматической локомотивной сигнализации
3. **Эксплуатация систем КТСМ** — содержание и обслуживание комплексов технического диагностирования
4. **Эксплуатация систем ЭЦ** — содержание и обслуживание электрической централизации

### Ремонтные функции
1. **Текущий ремонт** — плановый ремонт устройств СЦБ
2. **Капитальный ремонт** — полная замена оборудования
3. **Модернизация** — модернизация устройств СЦБ

### Контрольные функции
1. **Контроль технического состояния** — мониторинг состояния устройств
2. **Контроль работы устройств** — контроль работы устройств СЦБ
3. **Анализ отказов** — анализ причин отказов устройств

## Организационная структура

### Руководство
- **Начальник дистанции СЦБ** — возглавляет дистанцию, принимает ключевые решения
- **Заместитель начальника по эксплуатации** — курирует эксплуатационную работу
- **Заместитель начальника по ремонту** — курирует ремонтную работу
- **Главный инженер** — отвечает за техническое состояние

### Технический персонал
- **Старшие электромеханики** — руководят бригадами
- **Электромеханики СЦБ** — обслуживают и ремонтируют устройства СЦБ
- **Электромонтеры СЦБ** — выполняют работы по обслуживанию устройств
- **Операторы постов централизации** — управляют стрелками и сигналами

### Штаб дистанции
- **Начальник штаба** — организует планирование и документацию
- **Диспетчер дистанции** — координирует работу бригад
- **Инженер по технике безопасности** — контролирует соблюдение ТБ

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об отказе устройств СЦБ
- Оценка влияния на движение
- Определение необходимых мер

### 2. Выдвижение к месту ЧС
- Формирование аварийных бригад
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Оценка обстановки
- Осмотр места отказа
- Определение объема работ
- Разработка плана восстановления

### 4. Проведение восстановительных работ
- Устранение отказа
- Восстановление работоспособности устройств
- Проверка качества работ

### 5. Завершение работ
- Открытие движения
- Составление отчета
- Анализ причин отказа
