# НЦОП — Центр охраны окружающей среды

## Общее описание

Центр охраны окружающей среды (НЦОП) — подразделение, осуществляющее контроль за соблюдением экологических требований на железнодорожном транспорте. НЦОП обеспечивает экологическую безопасность деятельности железнодорожного транспорта и минимизацию негативного воздействия на окружающую среду.

## Основные функции

### Контрольные функции
1. **Контроль за выбросами** — контроль за выбросами загрязняющих веществ
2. **Контроль за сбросами** — контроль за сбросами сточных вод
3. **Контроль за отходами** — контроль за обращением с отходами
4. **Контроль за шумом** — контроль за уровнем шума

### Аналитические функции
1. **Мониторинг окружающей среды** — мониторинг состояния окружающей среды
2. **Анализ экологической обстановки** — анализ экологической обстановки
3. **Прогнозирование экологических рисков** — прогнозирование возможных экологических нарушений

### Организационные функции
1. **Разработка экологических мероприятий** — разработка мероприятий по охране окружающей среды
2. **Координация экологической деятельности** — координация экологической деятельности
3. **Взаимодействие с контролирующими органами** — взаимодействие с Росприроднадзором

## Организационная структура

### Руководство
- **Начальник НЦОП** — возглавляет центр, принимает ключевые решения
- **Заместитель начальника** — замещает начальника, курирует контрольную работу
- **Главный эколог** — отвечает за экологическую политику

### Отдел контроля
- **Начальник отдела контроля** — руководит контрольной работой
- **Инспекторы** — проводят проверки
- **Лаборанты** — проводят лабораторные исследования

### Отдел анализа
- **Начальник отдела анализа** — руководит аналитической работой
- **Аналитики** — анализируют экологическую обстановку
- **Специалисты по мониторингу** — проводят мониторинг

### Отдел экологических мероприятий
- **Начальник отдела** — разрабатывает экологические мероприятия
- **Специалисты** — разрабатывают мероприятия

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об экологическом нарушении
- Оценка масштаба нарушения
- Определение необходимых мер

### 2. Выдвижение к месту ЧС
- Формирование группы специалистов
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Оценка экологической обстановки
- Отбор проб
- Проведение анализа
- Оценка масштаба загрязнения

### 4. Ликвидация последствий
- Разработка плана ликвидации
- Координация работ по ликвидации
- Контроль хода работ

### 5. Завершение работ
- Проверка завершения работ
- Ведение документации
- Анализ причин нарушения
