# ДИ — Московская дирекция инфраструктуры

## Общее описание

Московская дирекция инфраструктуры (ДИ) — структурное подразделение ОАО «РЖД», осуществляющее эксплуатацию, обслуживание и развитие инфраструктуры Московской железной дороги. ДИ обеспечивает бесперебойную и безопасную работу железнодорожного пути, искусственных сооружений, устройств электроснабжения и связи.

## Основные функции

### Эксплуатационные функции
1. **Эксплуатация железнодорожного пути** — содержание и обслуживание пути
2. **Эксплуатация искусственных сооружений** — содержание мостов, тоннелей, путепроводов
3. **Эксплуатация устройств электроснабжения** — содержание контактной сети и тяговых подстанций
4. **Эксплуатация устройств связи и СЦБ** — содержание систем связи и автоматики

### Ремонтные функции
1. **Текущий ремонт пути** — плановый ремонт железнодорожного пути
2. **Капитальный ремонт пути** — полная замена верхнего строения пути
3. **Ремонт искусственных сооружений** — ремонт мостов, тоннелей
4. **Ремонт устройств электроснабжения** — ремонт контактной сети и подстанций

### Развивающие функции
1. **Модернизация инфраструктуры** — внедрение новых технологий
2. **Реконструкция объектов** — реконструкция станций и перегонов
3. **Увеличение пропускной способности** — развитие инфраструктуры

## Организационная структура

### Руководство
- **Директор ДИ** — возглавляет дирекцию, принимает ключевые решения
- **Первый заместитель директора** — замещает директора, курирует эксплуатацию
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническое состояние инфраструктуры

### Дистанции пути
- **Начальники дистанций пути (ПЧ)** — руководят дистанциями пути
- **Дорожные мастера** — руководят бригадами монтеров пути
- **Монтеры пути** — выполняют работы по содержанию пути

### Дистанции электроснабжения
- **Начальники дистанций электроснабжения (ЭЧ)** — руководят дистанциями
- **Старшие электромеханики** — руководят бригадами
- **Электромеханики** — обслуживают устройства электроснабжения

### Дистанции сигнализации, централизации и блокировки
- **Начальники дистанций СЦБ (ШЧ)** — руководят дистанциями
- **Старшие электромеханики** — руководят бригадами
- **Электромеханики СЦБ** — обслуживают устройства СЦБ

### Дистанции связи
- **Начальники дистанций связи (ЛЧ)** — руководят дистанциями
- **Старшие инженеры связи** — руководят бригадами
- **Инженеры связи** — обслуживают устройства связи

## Техническое оснащение

### Путевая техника
- **Путеукладочные краны** — УК-25, УК-25/9-18
- **Выправочно-подбивочно-отделочные машины** — ВПО-3000, ВПР-02
- **Балластоочистительные машины** — БМ, СЧ-600
- **Щебнеочистительные машины** — ЩОМ-4, ЩОМ-6
- **Рельсосварочные поезда** — ПРСМ

### Техника для электроснабжения
- **Вагоны-лаборатории контактной сети** — ВЛКС
- **Вышки контактной сети** — ВКС
- **Автодрезины** — для обслуживания контактной сети
- **Измерительные комплексы** — для диагностики контактной сети

### Техника для СЦБ и связи
- **Вагоны-лаборатории СЦБ** — для диагностики устройств
- **Автомобили техпомощи** — для оперативного выезда
- **Измерительные приборы** — для проверки устройств

## Порядок действий при ЧС

### 1. Получение информации о ЧС
- Принятие сигнала об аварии или нарушении
- Оценка масштаба повреждений
- Определение необходимых сил и средств

### 2. Выдвижение к месту ЧС
- Формирование аварийных бригад
- Выдвижение к месту происшествия
- Установление связи с диспетчером

### 3. Оценка обстановки
- Осмотр места повреждения
- Определение объема работ
- Разработка плана восстановления

### 4. Проведение восстановительных работ
- Устранение повреждений
- Восстановление работоспособности устройств
- Проверка качества работ

### 5. Завершение работ
- Открытие движения
- Составление отчета
- Анализ причин повреждения

## Взаимодействие с другими подразделениями

### ДАВС — Дирекция аварийно-восстановительных средств
- Координация аварийно-восстановительных работ
- Обмен информацией о повреждениях
- Совместное планирование мероприятий

### ДЦУП — Диспетчерский центр управления перевозками
- Получение информации о повреждениях
- Координация движения при ремонтах
- Согласование «окон» в движении

### НТЭ — Дирекция по энергообеспечению
- Координация работ по электроснабжению
- Взаимодействие при ремонте контактной сети
- Обеспечение безопасности при работах под напряжением

### ЦСС — Центральная станция связи
- Координация работ по устройствам связи
- Взаимодействие при ремонте
- Обеспечение связи

### СЦБ — Устройства сигнализации, централизации и блокировки
- Координация работ по устройствам СЦБ
- Взаимодействие при ремонте
- Обеспечение безопасности движения

## Нормативные документы

1. **Правила технической эксплуатации железных дорог РФ** — основные требования к эксплуатации
2. **Инструкция по текущему содержанию пути** — определяет порядок содержания пути
3. **Инструкция по содержанию контактной сети** — определяет порядок содержания КС
4. **Положение о системе технического обслуживания и ремонта** — определяет порядок ТОиР

## Показатели эффективности

1. **Состояние инфраструктуры** — оценка технического состояния
2. **Выполнение плана ремонтов** — процент выполненных ремонтных работ
3. **Количество отказов** — снижение количества отказов устройств
4. **Пропускная способность** — обеспечение требуемой пропускной способности
