# ДАВС — Дирекция аварийно-восстановительных средств

## Общее описание

Дирекция аварийно-восстановительных средств (ДАВС) — структурное подразделение, осуществляющее руководство и координацию аварийно-восстановительных работ на железнодорожном транспорте. ДАВС обеспечивает готовность аварийно-восстановительных сил и средств к ликвидации последствий аварий и чрезвычайных ситуаций.

## Основные функции

### Оперативные функции
1. **Координация аварийно-восстановительных работ** — руководство действиями всех служб при авариях
2. **Управление аварийно-восстановительными поездами** — диспетчеризация АВП
3. **Мобилизация сил и средств** — привлечение необходимых ресурсов при ЧС
4. **Оперативное планирование** — разработка планов ликвидации аварий

### Плановые функции
1. **Планирование аварийно-восстановительных мероприятий** — разработка планов на различные сценарии
2. **Обучение и тренировки** — подготовка персонала к действиям при ЧС
3. **Техническое оснащение** — обеспечение необходимой техникой и оборудованием
4. **Анализ аварийности** — анализ причин аварий и разработка мер по их предотвращению

## Организационная структура

### Руководство
- **Директор ДАВС** — возглавляет дирекцию, принимает ключевые решения
- **Первый заместитель директора** — замещает директора, курирует оперативную работу
- **Заместители директора** — курируют отдельные направления деятельности
- **Главный инженер** — отвечает за техническое состояние техники

### Оперативный отдел
- **Начальник оперативного отдела** — руководит оперативной работой
- **Старший диспетчер** — координирует действия аварийно-восстановительных поездов
- **Диспетчер АВП** — управляет аварийно-восстановительными поездами
- **Оперативный дежурный** — ведет оперативную обстановку

### Технический отдел
- **Начальник технического отдела** — руководит технической службой
- **Инженер по спецтехнике** — обслуживает аварийно-восстановительную технику
- **Механик** — выполняет ремонтные работы

### Отдел планирования
- **Начальник отдела планирования** — разрабатывает планы ликвидации аварий
- **Плановик** — составляет графики и планы работ
- **Аналитик** — анализирует аварийность

## Техническое оснащение

### Аварийно-восстановительные поезда
- **АВП-1, АВП-2, ...** — аварийно-восстановительные поезда различных типов
- **Восстановительные поезда** — специализированные поезда для восстановления пути
- **Мостопоезда** — для восстановления искусственных сооружений
- **Путевые машинные станции** — для выполнения путевых работ

### Специальная техника
- **Путеукладочные краны** — УК-25, УК-25/9-18 и другие
- **Автокраны** — подъемные работы
- **Экскаваторы и бульдозеры** — земляные работы
- **Мотовозы и дрезины** — доставка персонала и материалов

### Оборудование связи
- **Радиостанции** — оперативная связь
- **Телефоны** — проводная связь
- **Спутниковая связь** — связь в труднодоступных районах

## Порядок действий при ЧС

### 1. Получение сигнала об аварии
- Принятие сигнала от диспетчера или других служб
- Оценка масштаба аварии
- Определение необходимых сил и средств

### 2. Принятие решения
- Выбор оптимального плана ликвидации аварии
- Определение состава аварийно-восстановительных сил
- Назначение руководителя работ на месте

### 3. Мобилизация сил и средств
- Выдвижение аварийно-восстановительных поездов
- Привлечение дополнительной техники и персонала
- Установление связи с местом аварии

### 4. Координация работ
- Контроль хода восстановительных работ
- Координация действий всех служб
- Корректировка плана работ при необходимости

### 5. Завершение работ
- Проверка качества выполненных работ
- Открытие движения поездов
- Составление отчета о ликвидации аварии

## Взаимодействие с другими подразделениями

### ДЦУП — Диспетчерский центр управления перевозками
- Получение информации об авариях
- Координация движения поездов
- Согласование времени открытия движения

### МЧС — Министерство по ЧС
- Совместная работа при ликвидации последствий
- Взаимодействие при спасении пострадавших
- Координация действий в зоне ЧС

### МАБ — Медицинская выездная бригада
- Обеспечение медицинской помощи пострадавшим
- Эвакуация пострадавших
- Взаимодействие при травмах

### НТЭ — Дирекция по энергообеспечению
- Восстановление электроснабжения
- Координация работ на контактной сети
- Обеспечение безопасности при работах под напряжением

### СЦБ — Устройства сигнализации, централизации и блокировки
- Восстановление устройств СЦБ
- Координация работ на путях
- Обеспечение безопасности движения

### ЦЧС — Ситуационный центр мониторинга ЧС
- Передача информации о ходе работ
- Получение аналитических материалов
- Координация действий

## Нормативные документы

1. **Инструкция по ликвидации аварийных ситуаций** — определяет порядок действий при авариях
2. **Регламент взаимодействия служб** — определяет порядок взаимодействия с другими службами
3. **План ликвидации аварий** — разрабатывается для каждого участка
4. **Правила технической эксплуатации** — определяют требования к технике безопасности

## Показатели эффективности

1. **Время прибытия АВП к месту аварии** — не более 30 минут для близлежащих участков
2. **Время восстановления движения** — зависит от масштаба аварии
3. **Качество выполненных работ** — соответствует нормативным требованиям
4. **Готовность к ЧС** — постоянная готовность к действиям
