# Техника ДЦУП — диспетчерский центр управления перевозками

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДЦУП (Диспетчерский центр управления перевозками) для управления перевозками в регионе, мониторинга и корректировки графиков.

## Назначение

Управление перевозками в регионе, мониторинг, корректировка графиков.

## Технические средства

### Видеостена
- Размер: 4x4 м
- Количество экранов: 16 шт. Samsung 55"
- Назначение: отображение всей сети региона
- Разрешение: 4K

### АРМ диспетчера
- Количество мониторов: 5
- Программное обеспечение:
  - "Движение-2025" — система управления перевозками
  - Модуль мониторинга
  - Модуль корректировки графиков
- Резервное питание: ИБП на 8 часов

### Системы ИИ-аналитики
- Функции:
  - Прогнозирование задержек
  - Оптимизация маршрутов
  - Анализ загрузки сети
- Интеграция с системами управления

### GPS-трекинг локомотивов
- Обновление данных: в реальном времени
- Отображение на видеостене и АРМ диспетчера
- Интеграция с системами управления движением

### Резервные каналы связи
- Спутниковые терминалы Inmarsat
- Радиорелейные станции
- Оптоволоконные линии с резервированием

### Резервное оборудование
- Источник бесперебойного питания (ИБП) на 8 часов
- Дублирующие рабочие места
- Резервные серверы

## Примеры применения

**15 сентября 2024, перегон "Одинцово — Баковка"**
Сход состава. ДЦУП остановил движение, направил бригады — восстановление за 12 часов.

При сходе состава диспетчер:
- Остановил движение на участке
- Направил аварийно-восстановительные бригады
- Перенаправил поезда по альтернативным маршрутам
- Координировал работы по восстановлению движения

## Взаимодействие с другими подразделениями

Техника подразделения ДЦУП взаимодействует с техникой других подразделений при:
- Оперативном управлении с ЦД (Центральная дирекция управления движением)
- Координации с ЦУП (Центр управления перевозками Центральной дирекции)
- Работе с ДЦС (Центр организации работы станций)
- Аварийных ситуациях с ЦЧС (Ситуационный центр мониторинга ЧС)
- Взаимодействии с МЧС
