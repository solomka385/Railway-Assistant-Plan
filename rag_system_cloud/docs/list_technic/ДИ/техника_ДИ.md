# Техника ДИ — Московская дирекция инфраструктуры

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДИ (Московская дирекция инфраструктуры) для содержания и ремонта путей, мостов, тоннелей, контактной сети.

## Назначение

Содержание и ремонт путей, мостов, тоннелей, контактной сети.

## Технические средства

### Путеукладчики ПУ-4
- Назначение: замена рельсовых плетей
- Характеристики:
  - Укладка рельсов длиной до 25 м
  - Самоходное шасси
  - Гидравлический привод

### Машины для очистки балласта ЩОМ-Д
- Производительность: 600 м/час
- Ширина очистки: 3,5 м
- Назначение: очистка балласта от грязи и изношенных частиц

### Автовышки АГП-28
- Назначение: ремонт контактной сети
- Характеристики:
  - Высота подъёма: до 28 м
  - Грузоподъёмность: до 300 кг
  - На базе автомобильного шасси

### Дефектоскопы УЗД-77
- Назначение: контроль рельсов на наличие дефектов
- Тип: ультразвуковой
- Мобильность: переносные и стационарные

### Мобильные лаборатории
- База: ГАЗель NEXT
- Назначение: анализ качества балласта
- Оборудование:
  - Спектрометр для анализа металла
  - Оборудование для анализа грунта
  - Средства отбора проб

### Балластоукладчики БУМ
- Назначение: укладка нового балласта после замены рельсов
- Характеристики:
  - Шнековый механизм подачи
  - Равномерная укладка щебня

## Технические характеристики

### Регламент эксплуатации путеукладчика ПУ-4

**Перед работой:**
- Проверка захватов
- Проверка тормозной системы
- Проверка гидравлики
- Проверка систем безопасности

**Во время работы:**
- Соблюдение правил техники безопасности
- Контроль за правильностью укладки
- Проверка крепления рельсов

**После работы:**
- Очистка механизмов
- Проверка на наличие повреждений
- Запись в журнал

### Регламент эксплуатации машины ЩОМ-Д

**Перед работой:**
- Проверка фильтров
- Проверка транспортёра
- Проверка уровня масла
- Проверка систем безопасности

**Во время работы:**
- Работа только на закрытом перегоне
- Контроль за качеством очистки
- Проверка состояния балласта

**После работы:**
- Очистка механизмов
- Проверка износа рабочих органов
- Запись в журнал

### Регламент эксплуатации автовышки АГП-28

**Перед работой:**
- Проверка гидравлической системы
- Проверка тормозов
- Проверка систем безопасности
- Проверка изоляции (при работе с контактной сетью)

**Во время работы:**
- Соблюдение правил работы на высоте
- Использование средств индивидуальной защиты
- Контроль за безопасностью окружающих

**После работы:**
- Сложение стрелы
- Проверка на наличие повреждений
- Запись в журнал

## Регламент эксплуатации и обслуживания

### Путевые машины
- Ежедневная проверка технического состояния
- Еженедельное техническое обслуживание
- Плановое ТО — каждые 250 моточасов
- Капитальный ремонт — по износу

### Дефектоскопы
- Ежедневная калибровка перед работой
- Еженедельная проверка работоспособности
- Ежемесячное техническое обслуживание
- Поверка — ежегодно

### Мобильные лаборатории
- Ежедневная проверка оборудования
- Еженедельная калибровка приборов
- Ежемесячное техническое обслуживание
- Поверка — по графику

## Примеры применения

**14 августа 2024, перегон "Одинцово — Баковка"**
Обрыв контактной сети. Восстановлено за 3 часа бригадами ДИ.

При обрыве контактной сети были задействованы:
- Автовышка АГП-28 для доступа к месту обрыва
- Мобильная лаборатория для анализа состояния проводов
- Бригада для восстановления контактной сети

## Взаимодействие с другими подразделениями

Техника подразделения ДИ взаимодействует с техникой других подразделений при:
- Плановых ремонтах с ДРП (Московская дирекция по ремонту пути)
- Энергообеспечении с НТЭ (Московская дирекция по энергообеспечению)
- Диагностике с ЦТО (Производственный участок мониторинга и диагностики)
- Координации работ с ДЦС (Центр организации работы станций)

6. Инструкция по технике безопасности при ремонте контактной сети
7. Правила содержания железнодорожного пути
