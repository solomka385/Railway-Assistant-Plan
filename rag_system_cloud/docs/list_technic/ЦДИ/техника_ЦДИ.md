# Техника ЦДИ — Центральная дирекция инфраструктуры

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦДИ (Центральная дирекция инфраструктуры) для управления инфраструктурой сети РЖД.

## Назначение

Управление инфраструктурой сети РЖД.

## Технические средства

### Системы мониторинга инфраструктуры
- Характеристики:
  - Мониторинг состояния путей
  - Мониторинг контактной сети
  - Мониторинг искусственных сооружений
- Назначение: контроль состояния инфраструктуры

### Диагностическое оборудование
- Оборудование:
  - Дефектоскопы
  - Георадары
  - Нивелиры
  - Спектрометры
- Назначение: диагностика инфраструктуры

### Системы планирования ремонтов
- Характеристики:
  - Планирование ремонтных работ
  - Учёт ресурсов
  - Контроль выполнения
- Назначение: планирование и контроль ремонтов

### Мобильные лаборатории
- База: ГАЗель NEXT, УАЗ "Патриот"
- Оборудование:
  - Приборы для анализа
  - Средства измерения
  - Оборудование для отбора проб
- Назначение: оперативный анализ состояния инфраструктуры

### Системы связи
- Характеристики:
  - Радиостанции
  - Спутниковые терминалы
  - Мобильные телефоны
- Назначение: связь с подразделениями

### Специальное оборудование
- Путевые машины
- Оборудование для ремонта
- Средства измерения
- Оборудование для работы с документами

## Технические характеристики

### Регламент эксплуатации систем мониторинга

**Ежедневно:**
- Проверка работоспособности систем
- Анализ данных за сутки
- Проверка систем хранения

**Еженедельно:**
- Полный анализ данных
- Проверка алгоритмов анализа
- Тестирование систем оповещения

**Ежемесячно:**
- Обновление программного обеспечения
- Проверка систем хранения данных
- Техническое обслуживание

### Регламент эксплуатации диагностического оборудования

**Ежедневно:**
- Проверка работоспособности приборов
- Проверка зарядки аккумуляторов
- Калибровка перед использованием

**Еженедельно:**
- Полная проверка оборудования
- Проверка точности измерений
- Тестирование систем передачи данных

**Ежемесячно:**
- Техническое обслуживание
- Калибровка — ежеквартально
- Поверка — ежегодно

### Регламент эксплуатации мобильных лабораторий

**Перед выездом:**
- Проверка технического состояния автомобиля
- Проверка работоспособности оборудования
- Проверка зарядки аккумуляторов

**Во время работы:**
- Отбор проб по методике
- Анализ проб на месте
- Запись данных

**После работы:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

## Регламент эксплуатации и обслуживания

### Системы мониторинга
- Ежедневная проверка работоспособности
- Еженедельный анализ данных
- Ежемесячное обновление ПО
- Техническое обслуживание — ежемесячно

### Диагностическое оборудование
- Ежедневная проверка работоспособности
- Еженедельная полная проверка
- Ежемесячное ТО
- Калибровка — ежеквартально

### Мобильные лаборатории
- Ежедневная проверка технического состояния
- Плановое ТО — каждые 5000 км
- Калибровка приборов — ежеквартально

## Примеры применения

**Планирование капитального ремонта пути**
ЦДИ спланировала ремонт 100 км пути с использованием систем мониторинга и диагностического оборудования.

При планировании капитального ремонта были задействованы:
- Системы мониторинга для анализа состояния пути
- Диагностическое оборудование для детальной диагностики
- Системы планирования ремонтов для составления графика
- Мобильные лаборатории для анализа состояния инфраструктуры

## Взаимодействие с другими подразделениями

Техника подразделения ЦДИ взаимодействует с техникой других подразделений при:
- Планировании ремонтов с ДИ (Московская дирекция инфраструктуры)
- Координации с ДРП (Московская дирекция по ремонту пути)
- Диагностике с ЦТО (Производственный участок мониторинга и диагностики)
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
