# Техника НЦОП — Центр охраны окружающей среды

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением НЦОП (Центр охраны окружающей среды) для контроля экологической безопасности при авариях.

## Назначение

Контроль экологической безопасности при авариях.

## Технические средства

### Мобильные лаборатории "ЭкоТест"
- База: УАЗ
- Оборудование:
  - Газоанализаторы
  - Спектрометры
  - Пробоотборники
  - Оборудование для анализа почвы, воды, воздуха
- Назначение: оперативный анализ экологической обстановки

### Дроны с газоанализаторами
- Характеристики:
  - Время полёта: до 40 мин
  - Радиус действия: до 10 км
  - Газоанализаторы для контроля выбросов
- Назначение: контроль выбросов в воздухе

### Сорбенты и боновые заграждения
- Характеристики:
  - Сорбенты для сбора нефти и нефтепродуктов
  - Боновые заграждения для локализации разливов
- Назначение: локализация разливов нефтепродуктов на воде и суше

### Дезинфекционные установки
- Характеристики:
  - Портативные установки
  - Химические и термические методы дезинфекции
- Назначение: обработка территории при загрязнениях

### GPS-логгеры
- Характеристики:
  - Автоматическая запись координат
  - Передача данных в реальном времени
- Назначение: отслеживание распространения загрязнений

### Специальное оборудование
- Комплекты для отбора проб
- Контейнеры для транспортировки проб
- Средства индивидуальной защиты
- Оборудование для анализа данных

## Технические характеристики

### Регламент эксплуатации мобильных лабораторий

**Перед выездом:**
- Проверка технического состояния автомобиля
- Проверка работоспособности оборудования
- Проверка зарядки аккумуляторов
- Калибровка датчиков

**Во время работы:**
- Отбор проб по методике
- Анализ проб на месте
- Запись данных

**После работы:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

**Техническое обслуживание:**
- Плановое ТО — каждые 5000 км
- Калибровка приборов — ежеквартально
- Поверка — ежегодно

### Регламент эксплуатации дронов с газоанализаторами

**Перед вылетом:**
- Проверка состояния дрона
- Проверка газоанализаторов
- Проверка заряда аккумуляторов
- Калибровка датчиков

**Во время полёта:**
- Контроль за качеством данных
- Проверка работоспособности оборудования

**После полёта:**
- Очистка оборудования
- Проверка на наличие повреждений
- Зарядка аккумуляторов

**Техническое обслуживание:**
- Плановое ТО — ежемесячно
- Калибровка газоанализаторов — ежеквартально

### Регламент эксплуатации дезинфекционных установок

**Перед использованием:**
- Проверка работоспособности установки
- Проверка наличия реагентов
- Проверка систем безопасности

**Во время работы:**
- Контроль за качеством дезинфекции
- Проверка систем безопасности

**После использования:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

## Примеры применения

**Опрокидывание цистерны с мазутом в Калужской области**
НЦОП локализовал разлив, предотвратил попадание в реку.

При опрокидывании цистерны с мазутом были задействованы:
- Мобильные лаборатории "ЭкоТест" для анализа загрязнения
- Дроны с газоанализаторами для контроля выбросов
- Сорбенты и боновые заграждения для локализации разлива
- GPS-логгеры для отслеживания распространения загрязнений

## Взаимодействие с другими подразделениями

Техника подразделения НЦОП взаимодействует с техникой других подразделений при:
- Аварийных ситуациях с МЧС
- Координации с Росприроднадзором
- Совместных операциях с ДАВС (Дирекция аварийно-восстановительных средств)
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
