# Техника МЧ — механизированная дистанция погрузочно-разгрузочных работ

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением МЧ (Механизированная дистанция погрузочно-разгрузочных работ) для погрузки, разгрузки и перемещения грузов при авариях.

## Назначение

Погрузка, разгрузка, перемещение грузов при авариях.

## Технические средства

### Фронтальные погрузчики Caterpillar 950M
- Характеристики:
  - Грузоподъёмность: 5 т
  - Объём ковша: 3.5 м³
  - Мощность двигателя: 180 л.с.
- Назначение: погрузка сыпучих материалов, перемещение грузов

### Автокраны КС-55713
- Характеристики:
  - Грузоподъёмность: 25 т
  - Вылет стрелы: до 30 м
  - Высота подъёма: до 35 м
- Назначение: подъём и перемещение тяжёлых грузов

### Экскаваторы Komatsu PC300
- Характеристики:
  - Масса: 30 т
  - Объём ковша: 1.2–1.6 м³
  - Мощность: 180 кВт
- Назначение: земляные работы, расчистка завалов
- Оснащение:
  - Гидравлическая система
  - Быстрая смена навесного оборудования (грейфер, гидромолот)

### Контейнеровозы
- Характеристики:
  - Грузоподъёмность: до 40 т
  - Количество контейнеров: 1–2
- Назначение: перевозка контейнеров с оборудованием

### Вилочные погрузчики
- Характеристики:
  - Грузоподъёмность: 1–5 т
  - Высота подъёма: до 6 м
- Назначение: работа на складах, погрузка паллетированных грузов

### Специальное оборудование
- Гидромолоты для разрушения
- Грейферы для захвата грузов
- Ковши различного объёма
- Вилы для паллет

## Технические характеристики

### Регламент эксплуатации фронтальных погрузчиков

**Ежедневно:**
- Проверка уровня масла и топлива
- Проверка состояния шин
- Проверка гидравлической системы
- Проверка тормозов

**Перед работой:**
- Проверка работоспособности ковша
- Проверка систем безопасности
- Проверка освещения и сигнализации

**Во время работы:**
- Соблюдение правил техники безопасности
- Контроль за нагрузкой
- Проверка состояния оборудования

**После работы:**
- Очистка оборудования
- Проверка на наличие повреждений
- Запись в журнал

**Техническое обслуживание:**
- ТО-1 — каждые 100 моточасов
- ТО-2 — каждые 500 моточасов
- Капитальный ремонт — по износу

### Регламент эксплуатации автокранов

**Ежедневно:**
- Проверка уровня масла и топлива
- Проверка состояния шин
- Проверка тормозов
- Проверка систем безопасности

**Перед подъёмом:**
- Проверка тормозов
- Проверка редуктора
- Проверка тросов
- Проверка ограничителей угла поворота

**Во время работы:**
- Работа только при наличии двух операторов
- Контроль за нагрузкой
- Проверка устойчивости крана

**Запрещён подъём груза с отклонением от вертикали более 15°**

**После использования:**
- Очистка
- Смазка подвижных узлов
- Проверка износа тросов

**При работе на уклоне обязательна фиксация анкерами**

**Запрещена работа при ветре >10 м/с**

### Регламент эксплуатации экскаваторов

## Примеры применения

**Завал тоннеля оползнем**
МЧ расчистила проход за 12 часов, вывезла 500 т грунта.

При завале тоннеля были задействованы:
- Экскаваторы Komatsu PC300 для расчистки грунта
- Фронтальные погрузчики Caterpillar 950M для погрузки грунта
- Контейнеровозы для вывоза грунта
- Автокраны для перемещения крупногабаритных грузов

## Взаимодействие с другими подразделениями

Техника подразделения МЧ взаимодействует с техникой других подразделений при:
- Аварийно-восстановительных работах с ДАВС (Дирекция аварийно-восстановительных средств)
- Совместных операциях с ABP (Аварийно-восстановительные работы)
- Ремонте с ДРП (Московская дирекция по ремонту пути)
- Аварийных ситуациях с МЧС
