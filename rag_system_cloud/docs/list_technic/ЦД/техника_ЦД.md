# Техника ЦД — Центральная дирекция управления движением

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦД (Центральная дирекция управления движением) для управления движением на всей сети РЖД.

## Назначение

Управление движением на всей сети РЖД.

## Технические средства

### Центральный диспетчерский пункт
- Расположение: Москва
- Оснащение:
  - Видеостена
  - АРМ диспетчера
  - Системы связи
  - Системы анализа данных
- Назначение: оперативное управление движением

### Системы управления движением
- Программное обеспечение:
  - "Экспресс"
  - "Орлан"
- Характеристики:
  - Управление движением поездов
  - Оптимизация графиков
  - Контроль за соблюдением расписания
- Назначение: управление движением

### GPS-трекинг всех поездов
- Характеристики:
  - Обновление в реальном времени
  - Отображение на карте
  - Интеграция с системами управления
- Назначение: мониторинг положения поездов

### Системы прогнозирования
- Характеристики:
  - На основе ИИ
  - Прогнозирование задержек
  - Оптимизация маршрутов
- Назначение: прогнозирование и оптимизация

### Резервные ЦП
- Расположение:
  - Санкт-Петербург
  - Екатеринбург
- Характеристики:
  - Полное дублирование основного ЦП
  - Автоматическое переключение
  - Резервные каналы связи
- Назначение: резервирование на случай ЧС

### Специальное оборудование
- Системы видеоконференцсвязи
- Системы документирования
- Средства связи
- Оборудование для работы с документами

## Примеры применения

**Массовые сбои в Центральной России**
ЦД перенаправил 50% поездов — коллапс предотвращён.

При массовых сбоях в Центральной России:
- Центральный диспетчерский пункт координировал работы
- Системы управления движением оптимизировали маршруты
- GPS-трекинг отслеживал положение поездов
- Системы прогнозирования анализировали ситуацию

## Взаимодействие с другими подразделениями

Техника подразделения ЦД взаимодействует с техникой других подразделений при:
- Оперативном управлении с ЦУП (Центр управления перевозками Центральной дирекции)
- Координации с ДЦУП (Диспетчерский центр управления перевозками)
- Работе с ЦЧС (Ситуационный центр мониторинга ЧС)
- Аварийных ситуациях с МЧС
