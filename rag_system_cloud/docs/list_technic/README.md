# Документация специальной техники железнодорожного транспорта

## Обзор

Данный раздел содержит полную документацию о специальной технике железнодорожного транспорта, её характеристиках, использовании и регламентах эксплуатации.

## Структура документации

### Основные документы

#### 1. [`техника_подробное_описание.md`](техника_подробное_описание.md)
**Содержание:**
- Подробное описание техники по категориям
- Технические характеристики
- Подразделения, эксплуатирующие технику
- Примеры использования
- Особенности эксплуатации

**Категории техники:**
- Крановое оборудование (КС-5371, КС-55713, РДК-25)
- Землеройная техника (Э-153, JCB 3CX, Т-170)
- Погрузочная техника (Амкодор-332, ТО-28)
- Транспортная техника (КАМАЗ-5320, АГВ, МПТ)
- Путевая техника (УК-25, ЩОМ-4, ВПР-02, Б-М)
- Техника для контактной сети (АГ-60, МП)
- Мостостроительная техника (МУ, КМ)
- Медицинский транспорт (реанимобили, санитарные автомобили)
- Средства связи (Motorola GP-340, Thuraya)
- Беспилотные системы (DJI Matrice 30T, Autel EVO Max 4T)

## Использование документации

### Для обучения персонала
1. Изучение [`техника_подробное_описание.md`](техника_подробное_описание.md) для понимания техники
2. Изучение [`регламенты_использования_техники.md`](регламенты_использования_техники.md) для понимания регламентов
3. Изучение [`матрица_взаимодействия_техники.md`](матрица_взаимодействия_техники.md) для понимания взаимодействия

### Для планирования операций
1. Использование [`техника_подробное_описание.md`](техника_подробное_описание.md) для определения необходимой техники
2. Использование [`матрица_взаимодействия_техники.md`](матрица_взаимодействия_техники.md) для определения взаимодействия
3. Использование [`../list_employee/сотрудники_подробное_описание.md`](../list_employee/сотрудники_подробное_описание.md) для определения необходимого персонала

### Для анализа эффективности
1. Использование [`сценарии_использования_техники.md`](сценарии_использования_техники.md) для анализа хронологии
2. Использование [`матрица_взаимодействия_техники.md`](матрица_взаимодействия_техники.md) для анализа взаимодействия
3. Использование [`регламенты_использования_техники.md`](регламенты_использования_техники.md) для сравнения с планами

## Обновление документации

Документация должна обновляться:
- При внедрении новой техники
- При изменении технических характеристик
- При изменении регламентов эксплуатации
- После крупных аварий для анализа опыта
- Не реже 1 раза в год для актуализации данных

## Контакты

За актуализацией документации отвечает:
- ДАВС — координация обновлений
- Каждое подразделение — актуализация своих разделов
