# Техника СЦБ — устройства сигнализации, централизации и блокировки

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением СЦБ (Устройства сигнализации, централизации и блокировки) для обеспечения безопасности движения поездов.

## Назначение

Обеспечение безопасности движения поездов.

## Технические средства

### Светофоры
- Характеристики:
  - LED-индикация
  - Резервное питание
  - Высокая видимость
- Назначение: регулирование движения поездов

### Электрические централизации
- Типы:
  - Система МЦ
  - Система ЭЦ
- Характеристики:
  - Автоматическое управление стрелками
  - Контроль за положением стрелок
  - Резервное питание
- Назначение: централизованное управление стрелками

### Автоматическая локомотивная сигнализация (АЛСН)
- Характеристики:
  - Контроль скорости
  - Передача сигналов на локомотив
  - Автоматическое торможение
- Назначение: контроль скорости движения

### Диспетчерская централизация
- Типы:
  - Система ДЦ
- Характеристики:
  - Централизованное управление
  - Контроль за состоянием объектов
  - Автоматическое переключение
- Назначение: диспетчерское управление

### Мобильные бригады
- Оснащение:
  - Диагностические сканеры
  - Запасные части
  - Инструменты
  - Средства связи
- Назначение: оперативное обслуживание и ремонт

### Специальное оборудование
- Системы контроля
- Средства измерения
- Оборудование для ремонта
- Средства связи

## Примеры применения

**Отказ СЦБ на станции**
Бригада восстановила работу за 1 час — движение не останавливалось.

При отказе СЦБ на станции:
- Мобильная бригада оперативно выехала на место
- Диагностические сканеры определили причину отказа
- Были выполнены ремонтные работы
- Движение было восстановлено без остановки

## Взаимодействие с другими подразделениями

Техника подразделения СЦБ взаимодействует с техникой других подразделений при:
- Координации с ДЦС (Центр организации работы станций)
- Управлении с ДЦУП (Диспетчерский центр управления перевозками)
- Техническом обслуживании с ЦТО (Производственный участок мониторинга и диагностики)
- Ремонте с ДИ (Московская дирекция инфраструктуры)
