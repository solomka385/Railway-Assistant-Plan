# Техника ДРП — Московская дирекция по ремонту пути

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ДРП (Московская дирекция по ремонту пути) для ремонта и замены верхнего строения пути.

## Назначение

Ремонт и замена верхнего строения пути.

## Технические средства

### Выправочно-подбивочные машины ВПР-1200
- Назначение: автоматическая выправка и подбивка балласта после ремонта или просадок
- Характеристики:
  - Лазерная навигация
  - Точность: ±0.5 мм
  - Производительность: до 1200 м/час
- Система автоматического контроля качества

### Щебнеочистительные машины ЩОМ-Д
- Назначение: очистка балласта от грязи и изношенных частиц
- Характеристики:
  - Производительность: 600 м/час
  - Ширина очистки: 3.5 м
- Система фильтрации и транспортировки

### Балластоукладчики БУМ
- Назначение: укладка нового балласта после замены рельсов
- Характеристики:
  - Шнековый механизм подачи
  - Равномерная укладка щебня
  - Контроль толщины слоя

### Рельсосварочные машины МСР-1
- Назначение: стыковая сварка рельсов
- Характеристики:
  - Тип сварки: контактная
  - Автоматический контроль качества сварки
  - Мобильность: на базе железнодорожного подвижного состава

### Путевые измерительные тележки
- Назначение: контроль геометрии пути
- Характеристики:
  - Измерение ширины колеи
  - Измерение уровня
  - Измерение шаблона
  - Автоматическая запись данных

### Путеукладчики ПУ-4
- Назначение: замена рельсовых плетей
- Характеристики:
  - Укладка рельсов длиной до 25 м
  - Самоходное шасси
  - Гидравлический привод

## Технические характеристики

### Регламент эксплуатации машины ВПР-1200

**Перед работой:**
- Калибровка лазеров
- Проверка гидравлики
- Проверка рельсовых захватов
- Проверка систем безопасности

**Во время работы:**
- Контроль за точностью выправки
- Проверка качества подбивки
- Соблюдение правил техники безопасности

**После работы:**
- Очистка механизмов
- Проверка на наличие повреждений
- Запись в журнал

**Запрещено:**
- Включение при посторонних предметах на пути
- Работа без ограждения места работ

### Регламент эксплуатации машины ЩОМ-Д

**Перед работой:**
- Проверка фильтров
- Проверка транспортёра
- Проверка уровня масла
- Проверка систем безопасности

**Во время работы:**
- Работа только на закрытом перегоне
- Контроль за качеством очистки
- Проверка состояния балласта

**После работы:**
- Очистка механизмов
- Проверка износа рабочих органов
- Запись в журнал

### Регламент эксплуатации балластоукладчика БУМ

**Перед работой:**
- Проверка равномерности подачи щебня
- Проверка состояния шнеков
- Проверка гидравлики

**Во время работы:**
- Контроль за толщиной слоя балласта
- Проверка равномерности укладки

**После работы:**
- Очистка механизмов
- Проверка на наличие повреждений
- Запись в журнал

**Запрещена работа при t<-25°C без прогрева**

### Регламент эксплуатации рельсосварочной машины МСР-1

**Перед работой:**
- Проверка сварочного оборудования
- Проверка систем охлаждения
- Проверка систем безопасности

**Во время работы:**
- Контроль качества сварки
- Проверка геометрии стыка
- Соблюдение правил техники безопасности

**После работы:**
- Очистка сварочных электродов
- Проверка на наличие повреждений
- Запись в журнал

## Регламент эксплуатации и обслуживания

### Путевые машины
- Ежедневная проверка технического состояния
- Еженедельное техническое обслуживание
- Плановое ТО — каждые 250 моточасов
- Капитальный ремонт — по износу

### Измерительное оборудование
- Ежедневная калибровка перед работой
- Еженедельная проверка работоспособности
- Ежемесячное техническое обслуживание
- Поверка — ежегодно

### Сварочное оборудование
- Ежедневная проверка перед работой
- Еженедельное техническое обслуживание
- Плановое ТО — каждые 100 сварок
- Капитальный ремонт — по износу

## Примеры применения

**После схода состава — 200 м пути повреждено**
ДРП заменила рельсы и шпалы за 8 часов.

При восстановлении пути после схода состава были задействованы:
- Путеукладчик ПУ-4 для замены рельсов
- Выправочно-подбивочная машина ВПР-1200 для выправки пути
- Щебнеочистительная машина ЩОМ-Д для очистки балласта
- Балластоукладчик БУМ для укладки нового балласта
- Рельсосварочная машина МСР-1 для сварки стыков

## Взаимодействие с другими подразделениями

Техника подразделения ДРП взаимодействует с техникой других подразделений при:
- Плановых ремонтах с ДИ (Московская дирекция инфраструктуры)
- Аварийно-восстановительных работах с ABP (Аварийно-восстановительные работы)
- Координации работ с ДЦС (Центр организации работы станций)
- Диагностике с ЦТО (Производственный участок мониторинга и диагностики)
