# Техника ЦСС — Центральная станция связи

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ЦСС (Центральная станция связи) для обеспечения связи на всей сети.

## Назначение

Обеспечение связи на всей сети.

## Технические средства

### Радиорелейные станции Р-415
- Характеристики:
  - Диапазон: 2–18 ГГц
  - Дальность: до 50 км
- Назначение: резервные каналы связи при обрыве оптоволокна

### Спутниковые терминалы Inmarsat
- Характеристики:
  - Переносные терминалы
  - Передача данных до 60 кбит/с
  - Голос, SMS
- Назначение: индивидуальная связь персонала в удалённых районах

### Мобильные АТС на КАМАЗ-6350
- Характеристики:
  - Восстановление телефонной связи
  - До 200 абонентов
  - Развёртывание: 20 мин
- Назначение: восстановление телефонной связи

### Оптоволоконные линии
- Характеристики:
  - Высокая скорость передачи
  - Резервирование
  - Защита от повреждений
- Назначение: основная линия связи

### Системы шифрования
- Характеристики:
  - Защита каналов связи
  - Стандарт шифрования
  - Автоматическое шифрование
- Назначение: защита передаваемых данных

### Специальное оборудование
- Коммутационное оборудование
- Системы усиления сигнала
- Средства измерения
- Оборудование для работы с документами

## Технические характеристики

### Регламент эксплуатации радиорелейных станций

**Ежедневно:**
- Проверка ориентации антенн
- Проверка уровня сигнала
- Проверка систем питания

**Еженедельно:**
- Полная проверка станций
- Тестирование систем резервирования
- Проверка систем передачи данных

**Ежемесячно:**
- Техническое обслуживание
- Калибровка — по необходимости
- Проверка систем охлаждения

**Автоматическое переключение на резерв**

### Регламент эксплуатации спутниковых терминалов

**Ежедневно:**
- Проверка батарей перед выездом
- Регистрация в сети
- Проверка работоспособности

**При потере сигнала:**
- Автоматический переход на УКВ-радиосвязь Sepura SRH3800

**Еженедельно:**
- Полная проверка терминалов
- Тестирование систем связи

### Регламент эксплуатации мобильных АТС

**Перед развёртыванием:**
- Проверка технического состояния
- Проверка коммутационного оборудования
- Проверка систем питания

**Развёртывание:**
- Время развёртывания: 20 мин
- Проверка коммутации
- Тестирование связи

**После использования:**
- Проверка состояния оборудования
- Запись в журнал

## Регламент эксплуатации и обслуживания

### Радиорелейные станции
- Ежедневная проверка работоспособности
- Еженедельная полная проверка
- Ежемесячное ТО
- Капитальный ремонт — по износу

### Спутниковые терминалы
- Ежедневная проверка работоспособности
- Еженедельное тестирование
- Ежемесячное ТО
- Замена аккумуляторов — по необходимости

### Мобильные АТС
- Ежедневная проверка технического состояния
- Плановое ТО — каждые 5000 км
- Текущий ремонт — по необходимости

### Оптоволоконные линии
- Ежедневный мониторинг состояния
- Еженедельная проверка качества связи
- Плановое обслуживание — по графику

## Примеры применения

**Обрыв кабеля на 100 км**
ЦСС восстановила связь за 1 час через радиорелейные станции.

При обрыве кабеля были задействованы:
- Радиорелейные станции Р-415 для восстановления связи
- Спутниковые терминалы для связи с удалёнными районами
- Мобильные АТС для восстановления телефонной связи
- Системы шифрования для защиты данных

## Взаимодействие с другими подразделениями

Техника подразделения ЦСС взаимодействует с техникой других подразделений при:
- Управлении с ЦТУ (Отдел технического управления сетями связи)
- Координации с ЦУТСС (Центр управления технологической сетью связи)
- Работе с ДЦС (Центр организации работы станций)
- Взаимодействии с ЦЧС (Ситуационный центр мониторинга ЧС)
