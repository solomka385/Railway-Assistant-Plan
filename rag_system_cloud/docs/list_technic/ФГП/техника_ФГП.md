# Техника ФГП «ВО ЖДТ России»

## Общее описание

Данный документ содержит описание специальной техники, используемой подразделением ФГП «ВО ЖДТ России» для физической охраны объектов и борьбы с диверсиями.

## Назначение

Физическая охрана объектов, борьба с диверсиями.

## Технические средства

### Патрульные автомобили Ford Transit
- Характеристики:
  - Бронезащита
  - Полный привод
  - Системы связи
- Назначение: патрулирование территории

### Травматическое оружие
- Тип: пистолеты ОСА
- Характеристики:
  - Травматическое действие
  - Дальность стрельбы: до 50 м
- Назначение: самооборона и задержание

### Дроны с ночным видением
- Характеристики:
  - Время полёта: до 40 мин
  - Радиус действия: до 10 км
  - Ночное видение
- Назначение: скрытое патрулирование

### Системы видеонаблюдения
- Характеристики:
  - ИИ-аналитика
  - Запись видео
  - Ночное видение
- Назначение: наблюдение за объектами

### Мобильные КПП
- Оснащение:
  - Биометрический контроль
  - Системы видеонаблюдения
  - Средства связи
- Назначение: контроль доступа на объекты

### Специальное оборудование
- Системы сигнализации
- Средства связи
- Оборудование для работы с документами
- Средства индивидуальной защиты

## Технические характеристики

### Регламент эксплуатации патрульных автомобилей

**Ежедневно:**
- Проверка технического состояния
- Проверка уровня топлива
- Проверка систем связи

**Перед выездом:**
- Проверка работоспособности систем
- Проверка бронезащиты
- Проверка систем связи

**После выезда:**
- Проверка состояния автомобиля
- Запись в журнал

**Техническое обслуживание:**
- Плановое ТО — каждые 5000 км
- Текущий ремонт — по необходимости
- Капитальный ремонт — по износу

### Регламент эксплуатации дронов

**Перед вылетом:**
- Проверка состояния дрона
- Проверка ночного видения
- Проверка заряда аккумуляторов

**Во время полёта:**
- Контроль за качеством видео
- Проверка работоспособности оборудования

**После полёта:**
- Очистка оборудования
- Проверка на наличие повреждений
- Зарядка аккумуляторов

**Техническое обслуживание:**
- Плановое ТО — ежемесячно
- Калибровка ночного видения — ежеквартально

### Регламент эксплуатации систем видеонаблюдения

**Ежедневно:**
- Проверка записи видео
- Проверка чистки объективов
- Проверка работоспособности ИИ-аналитики

**Еженедельно:**
- Полная проверка камер
- Проверка настроек распознавания
- Тестирование системы оповещения

**Ежемесячно:**
- Обновление прошивки камер

## Примеры применения

**Попытка подрыва рельсов**
Патруль задержал злоумышленников — катастрофа предотвращена.

При попытке подрыва рельсов были задействованы:
- Патрульные автомобили для патрулирования
- Дроны с ночным видением для скрытого наблюдения
- Системы видеонаблюдения для фиксации нарушения
- Мобильные КПП для контроля доступа

## Взаимодействие с другими подразделениями

Техника подразделения ФГП взаимодействует с техникой других подразделений при:
- Охране объектов с РЦБЗ (Московский региональный центр безопасности)
- Координации с ЦЧС (Ситуационный центр мониторинга ЧС)
- Работе с МВД
- Взаимодействии с БВС (Беспилотное воздушное судно)
